# 任务1 初识媒体

随着科技的快速发展及传播载体的日益多元，媒体在现代社会中扮演着愈发关键的角色，深刻影响着人们学习、社交、消费等生活与工作的各个层面。

# 1. 什么是媒体

“媒体（Media）”一词，其历史根源可追溯至拉丁文的“Medium”，意指处于两者之间的中介角色。它在信息流通中扮演着桥梁的角色，广泛涵盖了人们用以传递和接收信息的各类工具、渠道、平台、中介或技术手段，是文字、声音等信息得以传递的媒介。概括地说，媒体是存储信息的实体和传递信息的载体，是承载和传播信息的媒介。这一概念通常蕴含着三重含义：

(1) 传播媒体: 指在自然界中承担信息传递功能的实体。例如, 蜜蜂被视为传播花粉的媒体, 而风将蒲公英的种子吹向远方, 成为植物繁衍信息的传播媒体。这些自然界的实体在不经意间扮演了信息传递的角色, 展示了媒体在广义上的含义。  
（2）物理媒介：指用于存储信息的实体，如磁盘、磁带、纸张等。它们能够长期保存信息，供人们随时查阅和使用。这些物理媒介是信息存储的重要基础。

（3）逻辑载体：用以表述信息的逻辑形式或手段，包括文字、声音、图形、图像、动画以及视频等多种形式。通过不同的逻辑结构和表达方式，它们生动直观地呈现信息给受众，使信息传递更加高效和便捷。

# 2. 媒体的分类

从不同的划分维度出发，媒体可作如下分类：

# （1）按出现的顺序划分

媒体通常可划分为以下几类：报纸刊物为第一媒体、广播为第二媒体、电视为第三媒体、互联网为第四媒体、移动互联网为第五媒体。

# (2) 按表现形式划分

媒体可以分为平面媒体、电波媒体和网络媒体。平面媒体主要包括印刷类、非印刷类等；电波媒体主要包括音频广播、电视广告等；网络媒体主要包括新闻网站、社交媒体平台、流媒体平台等。

# （3）按技术角度划分

为了使表达更加准确清晰，国际电信联盟（ITU）从技术的角度将媒体划分为感觉媒体、表示媒体、显示媒体、存储媒体和传输媒体。

感觉媒体是直接作用于人类感官的媒体形式，如文字、图像、声音、气味等。

表示媒体是信息的编码形式，如JPEG、PNG等图像编码，ASCII、GB2312等文本编码，MP3、AAC等声音编码。

显示媒体是实现电信号与感觉媒体转换的设备，如键盘、麦克风等输入设备，显示器、打印机等输出设备。

存储媒体是存储数据的实体介质，如硬盘、光盘、U盘等。

传输媒体是信号传输的物理通道，如双绞线、光纤、无线电磁波等。

# 任务2 感知数字媒体

随着数字媒体技术的深入拓展、终端设备的全面升级，它广泛渗透于社交、广告、影视、科研、远程办公、智慧城市等领域，深刻影响着人们的工作与生活，成为推动时代发展的关键力量。

# 1. 什么是数字媒体

数字媒体是指以二进制形式记录、处理、传播和获取的信息载体，涵盖数字化的文字、图形、图像、声音、视频、动画等感觉媒体，以及表示这些媒体编码的逻辑媒体。简而言之，数字媒体是利用数字技术创建、存储、传播和展示的信息内容，以数字编码形式存在，并通过计算机等设备进行处理和呈现。

# 2. 数字媒体的组成

数字媒体由两大核心部分组成：

(1) 信息内容: 这些信息内容均采用二进制形式进行表示, 即利用 “0” “1” 的组合来进行编码和存储信息。  
(2) 媒介载体: 这些媒介能够存储并传播以二进制形式表示的信息。当数字化信息通过特定媒介进行传播时, 就形成了数字媒体。

在媒体分类中，数字媒体代表着信息内容表现形式的一种革新。所有信息都必须能够被数字化，即“0”“1”来表示。这种数字化的信息表现形式决定了信息存储和传播方式的多样性。不同的信息形式，如文字、图像、声音等，需要采用不同的存储和传播手段来满足其特定需求。

# 3. 数字媒体的分类

数字媒体作为现代信息技术的重要组成部分，其分类形式多样，我们可以从不同的角度对其进行划分。

# （1）时间属性

按时间属性，数字媒体可分成静止媒体和连续媒体。静止媒体是指内容不会随着时间而变化的数字媒体，如文本和图片。连续媒体是指内容随着时间而变化的数字媒体，如音频和视频。

# (2) 来源属性

按来源属性，数字媒体可分成自然媒体和合成媒体。其中，自然媒体是指客观世界存在的景物、声音等，经过专门的设备进行数字化处理后得到的数字媒体，如数码相机拍摄的照片、数字摄像机拍摄的视频等。合成媒体指的是以计算机为工具，采用特定符号、语言或算法表示的，由计算机生成的文本、音乐、语音、图像和动画等，如用动画制作软件制作的动画角色、用视频制作软件制作的视频等。

# （3）组成元素

按组成元素，数字媒体可以分成单一媒体和多媒体。单一媒体是指由单一信息载体组成的媒体，如纯文本文件。多媒体指的是多种信息载体的表现形式和传递方式，如视频、动画等。通常人们所提及的数字媒体，大多情况下指的是多媒体。

# （4）内容特征

相关专家从产业的角度，基于媒体内容特征对数字媒体进行了分类，将数字媒体划分为数字动漫、数字影音、网络游戏、数字学习、数字出版和数字展示6个内容领域。

# 4. 数字媒体的特征

数字媒体技术作为整合硬件与软件的综合系统，贯穿了“信息采集  $\rightarrow$  加工  $\rightarrow$  存储  $\rightarrow$  传输  $\rightarrow$  呈现”的全流程，其核心特征可精炼为八大维度。

# （1）数字化存储

所有信息，无论是文字、图像还是声音，均被转化为计算机可识别的“0”“1”代码，实现了高精度存储与无损传输。就像我们用手机拍照时，光线瞬间被捕捉并转化为数字信号，永久地存储在设备中。

# (2) 双向交互性

数字媒体突破了传统电视单向传播模式，让用户能够实时参与进来。无论是直播时发送的弹幕评论，游戏中的角色操控，还是虚拟现实设备中与环境的实时互动，都让用户感受到了前所未有的参与感和沉浸感。

# （3）趣味体验性

通过多媒体融合，数字媒体打造了沉浸式的娱乐体验。短视频平台的AI换脸特效、电子书中的互动动画、体感游戏的动作捕捉等技术，让用户在享受娱乐的同时，也能感受到科技的魅力。

# （4）多元集成性

数字媒体技术智能地整合了多种媒体形态，使得信息呈现更加丰富多彩。网页设计中的图文、视频和超链接，微课制作中的PPT、配音和动画，电商直播中的商品展示、即时聊天和优惠弹窗，都是多元集成的典型应用。

# （5）艺术表达性

数字工具为创意设计提供了无限可能。Photoshop 软件制作的海报、Premiere 剪辑的 Vlog 都是数字媒体艺术表达的实际体现，这些作品呈现出设计人员的创意构思。

# （6）即时响应性

数字媒体技术保证了信息的实时性。例如，直播推流采用毫秒级延迟传输协议，视频通话搭载智能降噪算法，游戏引擎支持高帧率渲染技术。这些关键技术确保了用户能够即时获取和响应信息，提升了用户体验的流畅性和真实性。

# （7）用户主权性

# 任务2 了解数字媒体作品设计原则

数字媒体作品设计需要遵循一系列原则，以确保作品在视觉、功能和用户体验等方面达到较高的水准。常见的设计原则有用户中心、交互性、兼容性、性能优化、可扩展性、安全性和合法性等原则。

# 1. 用户中心

用户中心原则强调在设计和开发过程中始终将用户的需求、习惯和体验放在首位。它要求通过用户研究、访谈、测试等方法深入了解目标用户的行为模式、痛点和期望，从而设计出符合用户实际使用场景的界面和功能。例如，界面布局应简洁直观，易于理解和操作，有助于用户快速找到所需信息和功能。同时，界面设计需考虑不同用户群体的多样性，确保不同年龄、不同技术水平的用户都能轻松上手。此外，系统应提供个性化选项，允许用户根据自身偏好调整界面和功能。

# 2. 交互性

交互性原则确保了用户与作品之间的有效沟通。设计者需要在用户首次接触某个功能时，提供明确的操作引导。这可以通过提示框或引导动画来实现，帮助用户快速了解操作流程。此外，当用户进行操作时，系统应立即给予反馈，明确告知用户操作是否成功。这种反馈可以是视觉上的，比如按钮点击后的颜色变化；听觉上的，如操作完成时的音效；甚至是触觉上的，例如手机的震动反馈。

# 3.兼容性

兼容性原则确保了数字媒体作品在不同设备、不同操作系统下都能够顺畅运行。在数字化时代，若数字媒体作品想扩大受众范围，就必须具备出色的兼容性。这意味着在硬件层面，数字媒体作品需确保在电脑、平板、手机等多样化终端设备上实现稳定运行，不因设备差异而出现显示偏差、画面卡顿等状况；在软件层面，无论是主流的Windows、iOS、Android操作系统，还是小众的操作系统，都需要进行针对性的适配测试，确保开发的数字媒体内容在不同的设备、操作系统、浏览器和屏幕尺寸上都能正常显示和运行，提升其传播效果和影响力。

# 4. 性能优化

性能优化原则是提升用户体验的关键。随着用户对响应速度和加载效率的要求不断提高，开发过程中可以通过优化算法和合理压缩等方法来减少资源占用。优化算法可以提高程序的执行效率，减少不必要的计算和操作，从而加快响应速度。合理压缩则可以减小文件大小，降低传输时间，提高加载速度。在实际开发中，应根据具体需求和项目特点，综合运用多种性能优化方法，以达到最佳的用户体验效果。

# 5. 可扩展性

可扩展性原则是数字媒体作品实现长远发展的关键。在架构设计时，应采用模块化和分层架构的方法，确保系统具备良好的可扩展性和维护性。模块化设计能够将复杂的系统分解为多个独立的模块，每个模块负责特定的功能，相互之间通过清晰的接口进行交互。分层架构则进一步优化了系统的组织方式，将系统划分为多个层次，每个层次都有特定的功能和角色，使得系统的结构更加清晰，易于理解和维护。

# 6. 安全性

在数字媒体作品开发过程中，采用高强度的数据加密技术对用户的个人信息、浏览记录、交互数据等进行加密处理是确保数据在传输和存储过程中安全的关键措施。例如，在用户注册登录环节，使用高强度加密算法对用户密码进行加密，这样即使数据在传输或存储中被窃取，攻击者也难以破解原始密码。此外，严格限制数据访问权限，设置不同的数据访问级别，确保只有授权人员才能访问特定敏感数据，可以有效防止内部人员随意获取和泄露数据。同时，开发人员还需要

建立完善的安全防护体系，抵御黑客攻击、恶意软件入侵等安全威胁，并利用专业工具定期对系统进行安全漏洞扫描，及时修复漏洞，以保障作品的稳定运行。

# 7. 合法性

合法性原则要求数字媒体作品开发需全流程遵循《著作权法》《个人信息保护法》及行业伦理规范，主要包括版权保护、隐私保护、内容合规三个方面。版权保护在数字媒体作品的创造、生产、传播、销售、使用等整个生命周期内，管理和保护数字媒体作品的版权，确保数字媒体作品的合法占有、使用、传播和管理，明确未经作者同意，他人不得擅自使用、更改或出版数字媒体作品。依据相关法律法规，数字媒体作品开发者必须严格保护用户隐私，泄露用户隐私属于违法行为，必将承担相应法律后果。此外，开发数字媒体作品还需要内容合规，坚决不传播危害国家安全、违背公序良俗或包含暴力色情、歧视性的内容，充分考虑数字媒体作品对社会和用户可能产生的影响，积极传播正能量和有益的信息，以维护健康的数字生态。

# 任务2 体验数字媒体作品的创作流程

数字媒体作品的创作流程一般可分为6个阶段：策划、设计、制作、测试评估、发布推广、反馈改进。该流程确保了数字媒体作品从策划到最终发布的顺利推进。

# 1. 策划阶段

策划阶段是数字媒体作品创作的首要环节，也是项目生命周期的关键阶段。此阶段需明确作品目标，精准定位核心内容与创作目的。同时，要深入分析目标受众的特征，包括年龄、性别、职业、兴趣爱好、消费习惯、文化背景等维度，通过市场调研、问卷调查、用户访谈等方式，了解其需求与期望。此外，还应制定详细的策划方案，构思作品的整体创意，为每个阶段设定具体目标和时间表，确保项目顺利推进。

# 2. 设计阶段

数字媒体作品的设计阶段是将创意转化为可操作方案的核心环节，主要包含视觉设计、交互设计与内容架构设计。视觉设计涵盖界面设计、图形图像设计、色彩搭配以及字体设计与排版，旨在打造有吸引力的视觉效果。交互设计聚焦于用户流程设计、交互元素设计以及用户体验优化，搭建用户与作品的沟通桥梁。内容架构设计则负责信息分类、组织及导航系统设计，尤其在内容繁多的作品中，能提升用户获取信息的效率，确保其高效理解与利用作品内容。

# 3. 制作阶段

数字媒体作品的制作阶段是将前期策划和设计转化为实际作品的关键环节，主要包括素材的制作与收集、工具与软件的选择以及作品的创作。在素材方面，需根据作品需求创作或收集图片、视频、音频等各类素材，来源包括网络、实地拍摄、原创绘制或购买版权等，但需确保版权合法。工具与软件的选择则依据作品类型和创作需求，如平面设计使用Adobe Photoshop、Adobe Illustrator，视频制作选用Adobe Premiere、剪映等，并确保设备性能达标。作品创作阶段，创作人员遵循脚本和策划方案，运用选定的工具对素材进行加工处理，如视频剪辑、特效添加、图像合成、文字排版等，同时不断审视作品，确保其符合主题和目标要求。

# 4. 测试评估阶段

数字媒体作品的测试评估阶段是确保作品质量与用户体验的关键步骤，包含作品测试和用户体验测试。作品测试需进行功能测试，确保作品功能正常且符合预期；兼容性测试，保证作品在不同设备和系统环境下稳定呈现；性能测试，评估作品运行速度、响应时间及资源占用等性能指标。用户体验测试则邀请真实用户试用作品，收集他们的反馈意见，观察用户在使用过程中的具体操作行为，了解他们对作品界面设计、交互流程、功能易用性等方面的感受和评价，以便发现作品存在的问题并加以改进，提升用户体验。

# 5. 发布推广阶段

数字媒体作品的发布推广阶段是将作品推向目标受众、实现价值最大化的关键环节，包含选择发布平台、制定推广策略、持续运营与更新。在选择发布平台时，需根据作品类型和目标受众特点，精心挑选合适的平台，如短视频作品选抖音、快手，学术类作品选学术网站等。推广策略方面，可结合社交媒体推广、搜索引擎优化和付费推广等多种方式，提高作品曝光度。此外，持续运营与更新也是发布推广的重要内容，需定期更新作品，保持用户的新鲜感和忠诚度，同时持续开展推广活动，拓展用户基础。

# 6.反馈改进阶段

数字媒体作品的反馈改进阶段是持续提升作品品质、超越用户期望的关键过程，主要包括收集反馈信息和实施改进与优化措施。收集反馈信息需通过多种渠道设置反馈入口，开展用户调研活动，全面收集用户、合作伙伴和行业专家的反馈。改进与优化则基于对反馈信息的深入分析，制定详尽计划。由开发团队修复功能漏洞、提升系统性能；由设计团队优化界面布局、色彩和交互元素；由内容团队审核更新内容，确保其准确、时效和吸引力。改进完成后，持续关注用户反馈，为后续改进提供依据。

# 任务 走进数字图像与色彩空间

数字化图像技术，作为现代信息处理领域的一项重要创新，通过将模拟图像转换为数字信号，实现了图像信息的精确记录、高效存储与灵活处理。它不仅极大地推动了图像处理的发展，还为图像传输、分析与应用提供了前所未有的便利。这一技术的广泛应用，标志着视觉信息时代的全面到来。

# 1. 什么是数字化图像技术

在计算机获取的视觉媒体中，图像占很大比例。其中，“图”指的是物体反射或透射光的分布，是客观世界的反应。而“像”则是人的视觉系统对图的响应，是人脑对图的印象或认识。“图”与“像”的有机结合，既展现了物体的客观存在，又体现了人类的主观感知。

数字化图像技术是指将传统的模拟图像转换为数字形式，并通过计算机进行存储、处理和传输的技术。其核心在于将连续的光学信号转化为离散的数字信号，使图像能够以二进制数据的形式被电子设备识别和操作。

# 2. 数字图像的类型

数字化图像技术不仅涉及将模拟图像转换为数字形式，还涵盖了如何存储和处理这些数字图像。根据存储方式和处理方式的不同，数字图像主要分为两大类：位图和矢量图。

# （1）位图

位图，亦称栅格图、像素图或点阵图，它由许多离散的、方格的点组成，类似于“行×列”的点阵，这些点称为像素。像素作为位图的基本组成单元，通过二进制数记录其位置、颜色等属性信息。当这些像素按照特定的规则排列组合时，便形成了位图图像。

位图文件的大小，主要取决于照片被分割成的小方块数量，即分辨率的高低。分辨率越高，文件体积也就越大。另一个重要因素则是颜色深度，又称灰度级，它反映了图像中每个像素所能表达的颜色种类或色彩的精细程度。颜色深度通常以“位（bit）”为单位来计量，例如，一个24位的颜色深度则能让每个像素表达出约1677（ $2^{24}$ ）万种不同颜色，这样的色彩范围让人类肉眼难以分辨出颜色之间的差异。位图图像数据容量的大小计算公式如下：

位图图像大小（B）=水平像素点数  $\times$  垂直像素点数  $\times$  颜色深度/8

例如一副分辨率为  $1024 \times 768$ ，采用 24 位真彩色的图像，其数据容量为：

$$
1 0 2 4 \times 7 6 8 \times 2 4 / 8 / 1 0 2 4 / 1 0 2 4 \approx 2. 2 5 (\mathrm {M B}) 。
$$

因此，位图文件通常占用较大的存储空间，非常适合展现丰富的色彩变化与细腻的色彩过渡效果。然而，当在屏幕上以较大倍数放大显示时，由于放大的是其中的每个像素点，位图可能会出现失真，变得模糊不清。如图3-1、3-2所示。

# 3.数字图像色彩空间

在视觉艺术与数字技术的交汇领域，色彩是数字图像中最具吸引力和表现力的核心元素。当光线与物体相互作用并发生反射时，这些反射光线被精准捕捉并转换成数字信号，从而使得色彩能够在数字图像中得到精确重现。色彩不仅是光的一种直观展现，更是数字化图像技术不可或缺的基石。

# (1) 色彩三要素

色彩的种类成千上万，除了无彩色以外，任何有彩色均有色相、明度和纯度上的变化，以此来区别各种不同的色彩，人们称它们为色彩的三要素或三属性。无论是探索自然界的绚丽色彩，还是在艺术创作与设计中运用色彩，都离不开对色彩三要素的深刻理解和灵活应用。

色相，又叫色度，是区分不同色彩的首要特征。简单来说，色相就是色彩的名称，在可见光谱的广阔范围内，红、橙、黄、绿、青、蓝、紫等每一种色相都对应着特定的波长与频率。它们有序地排列，从短到长，既展现出色彩的多样性，又保持着和谐的统一，如图3-5所示。

# (2) 色彩模式

色彩模式是数字世界中表示颜色的一种算法，它将某种颜色表现为数字形式的模型。常用的色彩模式有 RGB 模式、CMYK 模式、Lab 模式和 HSB 模式等。

RGB 模式是一种基于三原色混合的色彩模型，它依据色光的加法混合原理，通过红（Red）、绿（Green）、蓝（Blue）三种色光以不同的比例混合来生成各种颜色，如图 3-8 所示。R、G、B 三种成分的取值范围均为  $0 \sim 255$ ，能够组合出约 1670 万种颜色（即  $256 \times 256 \times 256$  种组合），几乎涵盖了人类视力所能感知的全部颜色范围，因此被称为真彩色。例如，RGB(255,0,0) 代表红色、RGB(0,255,0) 代表绿色、RGB(0,0,255) 代表蓝色。在显示器、数字摄影、图形设计和网页设计等领域，一般采用 RGB 模式。

CMYK 模式是一种在彩色印刷中广泛应用的色彩模式，它基于颜料三原色的配色原理，并加入了黑色油墨，利用四种色彩进行混合叠加，以达到全彩印刷的效果。其中，青色（Cyan，C）、品红色（Magenta，M）和黄色（Yellow，Y）被视为颜料三原色，如图 3-9 所示。CMYK 模式的 4 色使用  $0\% \sim 100\%$  的值，当 4 种色彩均为  $0\%$  时，就会产生纯白色：当 4 种色彩均为  $100\%$  时，则会得到纯黑色。

Lab 模式是一种色彩范围较广的色彩模式，包含 RGB 模式和 CMYK 模式中所有的色彩，它不依赖于光线或颜料，是与设备无关的颜色系统。它用亮度通道 L（0-100 整数）和两个色相通道 a（ $-128 \sim 127$  整数，代表绿到红）、b（ $-128 \sim$

127 整数，代表蓝到黄）来定义色彩，正为暖色，负为冷色。由于 Lab 模式的色彩一致性，它常用作 RGB 和 CMYK 等色彩模式转换的中间桥梁。

![](images/2bbf932b450a762b4f650cda4e2bdf8fa839b6aa767c3a018d93ab6c5de22711.jpg)

![](images/2306c42f9a25d0dd1d800685491e010aa5030a4f2df658203c86b67db1c2b666.jpg)  
图3-9颜料三原色

HSB 模式是一种更直观，更接近人们视觉原理的色彩模式，它采用颜色的三属性来表色，即将颜色的三属性进行量化，H 代表色相、S 代表饱和度、B 代表亮度。饱和度和亮度以百分比值（ $0\% \sim 100\%$ ）表示，色相以角度（ $0^{\circ} \sim 360^{\circ}$ ）表示。HSB 模式为将自然色彩转换为计算机创建的色彩提供了一种直接方法。因此，在进行色彩校正等图像处理工作时，HSB 模式经常被采用，以便精确设置和调整色彩的饱和度、亮度和色相。

# 4.数字图像处理技术

# （1）什么是数字图像处理技术

数字图像处理技术包含了一系列的操作和算法，用于对图像进行增强、恢复、分割、特征提取及识别等处理，旨在改善图像质量并提取有用信息，以满足各种应用需求。

# （2）数字图像处理技术的分类

数字图像处理技术可分为全局处理技术和局部处理技术两大类。

全局处理技术能够调整图像的整体效果，如优化亮度与对比度、校正色彩平衡及创意应用滤镜效果等。

局部处理技术则是对图像中特定局部进行精细修改，这通常涉及选择或定位需修改的区域，如使用仿制图章进行精准修复、涂抹工具实现平滑过渡及橡皮擦进行精确擦除等。

数字图像处理技术在众多领域展现出了广泛的应用价值。在医学影像领域，它提高了诊断的准确性和效率；在安防监控中，它增强了监控画面的清晰度和识别度；在自动驾驶技术中，它助力车辆精准识别路况；此外，在广告、艺术等领域，它也发挥着美化图像和创意设计的重要作用。

# 1.音频数字化

音频数字化是指将模拟音频信号转换为数字格式的过程，它不仅提升了音质保真度，还使得音频文件易于编辑、存储与分享。

# （1）声音信号数字化

声音信号的数字化包含三个核心步骤：采样、量化和编码。

采样是指通过每隔一定时间间隔选取信号样值，来替代原本在时间轴上连续的信号，从而实现模拟信号的离散化。如图3-11所示。

# （2）数字音频技术重要指标

采样频率：是指每秒钟采集声音样本的数量，其单位通常为赫兹（ $\mathrm{Hz}$ ）。采样频率的高低直接影响了声音信号的还原度和保真度。为了避免失真，采样频率应至少是原始声音信号最高频率的两倍，较低的采样频率会导致声音失真，甚至产生难以忍受的噪声。人耳能听到的最高频率大约为  $20\mathrm{kHz}$ 。

量化位数：决定了每个采样点可以表达的不同电平或幅度等级的数量。它直接影响了声音的动态范围和信噪比。量化位数越高，每个采样点能够表示的电平值就越多，声音的细节和层次感也就越丰富，音质也就越好。

声道数：是指录制或播放音频时所使用的独立声音通道的数量。最基础的配置是单声道，它仅含一个音频通道。立体声包含左、右两个声道，能模拟声音的空间和方向特性。随着技术的进一步发展，杜比全景声等沉浸式环绕声技术为人们提供了更为丰富听觉体验。

# （3）音频文件大小计算

音频文件的大小可以通过以下公式来计算：

音频大小（B）=采样频率（Hz）×量化位数（位）×声道数×时长（秒）/8

例如，以16位量化位数、双声道抓取一张60分钟的CD唱片保存为WAV波形文件时，所需要的存储空间为：

$$
4 4. 1 \times 1 0 0 0 \times 1 6 \times 2 \times 3 6 0 0 / 8 / 1 0 2 4 / 1 0 2 4 \approx 6 0 5. 6 (\mathrm {M B})
$$

# 2. 视频数字化

# （1）视频信号数字化

视频数字化是将模拟视频信号经过视频采集卡转换成数字视频文件保存在存储设备的过程。在使用时，将数字视频文件从存储设备中读出，再还原成为电视图像加以输出。

视频数字化主要应用在模拟视频占据主流的早起时代，随着数字技术的普及，数字摄像机已经能够直接生成数字信号，人们只需将它们转存至硬盘存储设备即可。无论是从模拟信号转换，还是直接接收数字信号，视频信号的采集和存储都需要大量空间及高速数据传输技术支持。为了确保视频数据的正常处理，通常在采集和播放过程中会对视频进行压缩和解压缩处理。

视频分辨率：是描述视频画面中每行和每列像素数量的参数，通常以“宽度  $\times$  高度”的形式表示，单位是像素（px），它是衡量视频清晰度的重要指

标。分辨率越大，视频画面中的细节越丰富，视觉体验越清晰，但也会带来更大的数据量，对存储和传输设备提出更高的要求。常见的分辨率标准如表3-2所示。

# 任务1 分析数字媒体压缩技术

数字媒体压缩技术是指通过减少数据冗余来降低文件大小，同时尽量保持媒体质量的技术。

# 1. 数字媒体压缩的原因

在数字世界中，数字媒体内容的文件体积往往十分巨大，这主要是因为它们包含了大量的详细信息。这些详细信息中常常掺杂着大量的冗余数据，导致文件体积过大，给存储和传输带来了很大的不便。这就需要通过数字媒体压缩技术来大幅度减小图像、声音和视频文件的体积，缓解存储空间被占用的压力，提高数据传输效率。

# 2. 数字媒体的数据冗余

数字媒体压缩技术的核心在尽可能不损害媒体质量的基础上，大幅度降低数据量，识别和移除其中存在的各类冗余信息。数据冗余主要有以下几种类型：

# （1）空间冗余

在静态图像的数字化处理过程中，相邻像素的数据往往高度相似或完全一致。例如，在一张包含大片蓝色天空的照片中，许多像素的颜色值都相同或非常接近。这种冗余信息可以通过帧内压缩技术去除，从而显著减小图像文件的大小，这就是空间冗余。

# （2）时间冗余

在动态图像中，相邻帧之间的图像内容往往存在大量重复或相似性。例如，在拍摄一个缓慢移动的物体时，背景可能保持不变，而物体本身只发生了微小的位移。这种图像序列中的相关性冗余被称为时间冗余。

# （3）视觉冗余

人类视觉系统对某些细节和变化并不敏感例如在高亮度背景下，视觉灵敏度会下降，对灰度值的细微变化可能无法察觉。因此，在不影响整体视觉效果的前提下，可以对这些不敏感区域的数据进行压缩或忽略。这种基于人类视觉系统特性的冗余被称为视觉冗余。

# （4）听觉冗余

人耳所能听到的声音信号频率范围有限，大约在  $20\mathrm{Hz}$  到  $20\mathrm{kHz}$  之间。在

这个频率范围之外的声音信号，即使被记录下来，也无法被人类感知。因此，在音频压缩过程中，可以安全地去除这些无法被人耳察觉的声音信号，从而减小音频文件的大小，这就是听觉冗余。

# （5）编码冗余

对于文本数据和程序来说，数据的信息量常常少于数据本身所呈现的形式。在一段包含大量重复字符的文本中，可以使用压缩编码来表示这些重复字符，从而减小数据量。例如，字母序列“AABBBCCCCDDDDDDD”可以用压缩编码“2A3B4C8D”来表示，原本需要17个字符的文本，通过编码就只需要8个字符来表示。这种由于数据表示方式而产生的冗余被称为编码冗余。

# （6）知识冗余

在某些情况下，数据的理解与先验知识密切相关。例如，在人脸识别任务中，人脸具有固定的结构特征，如眼睛、鼻子和嘴巴的位置关系，这些信息可以作为先验知识来指导数据压缩过程，从而去除那些不影响人脸识别结果但增加数据量的冗余信息。这种由于先验知识而产生的冗余被称为知识冗余。

# 3. 数字媒体压缩的分类

按媒体信息压缩前后是否有损失划分，数字媒体压缩可分为有损压缩和无损压缩。

# （1）有损压缩

有损压缩是一种在压缩过程中会丢失部分信息的压缩方法。这种丢失的信息在解压缩后无法恢复，因此解压缩后的信息与原始信息存在差异。有损压缩广泛应用于语音、图像和视频数据的压缩中，因为它能够提供较高的压缩比，从而显著减小文件的大小。常见的有损压缩格式包括JPEG、MP3、RM、RMVB、WMA和WMV等。

# (2) 无损压缩

无损压缩，是一种在压缩过程中不会丢失任何信息的压缩方法。因此，解压缩后的信息与原始信息是完全相同的。这种压缩方法通常用于需要保持数据完整性的场合，如文本数据、程序和特殊应用场合的图像数据（如指纹图像、医学图像等）的压缩。常见的无损压缩格式包括GIF、APE、FLAC、TAK、TTA和ZIP等。

# 4. 数字媒体压缩的标准

数字媒体压缩标准是指为了减少数字媒体文件（如图像、音频、视频等）的数据量，同时尽可能保持其质量而制定的一系列技术规范和算法。以下是常用的数字媒体压缩标准：

# (1) 静态图像压缩标准 (JPEG)

JPEG标准是由国际标准化组织（ISO）、国际电工委员会（IEC）以及国际电信联盟电信标准化部门（ITU-T）联合成立的联合图像专家组所制定的静态图像压缩标准。主要可分为三种类型：标准JPEG、渐进式JPEG以及JPEG2000。

标准JPEG在网页下载时，图像会按照由上至下的顺序逐行显示，直至下载完成才能展现完整图像；渐进式JPEG在下载过程中，会先快速呈现出图像的粗略外观，随后逐渐细化至完整内容；而JPEG2000作为JPEG的最新标准，展现出了极高的灵活性，它不仅能够根据需求采用不同的压缩比对图像进行压缩，还同时支持无损压缩和有损压缩。与早期标准相比，JPEG2000在有损压缩条件下也能呈现出更高品质的图像效果。它在医学图像分析、处理等对图像品质要求严苛的领域得到了广泛应用。

# (2) 动态图像压缩标准 (MPEG、H. 264)

动态图像压缩标准目前主要有MPEG标准和H.264标准。

MPEG标准运动图像专家组是由国际标准化组织和国际电工委员会联合成立的，负责开发电视图像数据和声音数据的编码、解码标准和它们的同步标准。该专家组开发的标准称为MPEG标准，包括MPEG-1、MPEG-2、MPEG-4、MPEG-7、MPEG-21等。MPEG-4是目前主流的动态图像压缩标准，它引入了基于对象的编码技术，能够分离背景和前景，从而实现更灵活的压缩和编辑，因此被广泛应用于网络视频流、移动设备和视频会议，典型文件格式包括.mp4、.avi和.mov。

H.264 标准是由国际电信联盟的视频编码专家组与运动图像专家组联合组成的联合视频组提出的视频压缩标准。它在MPEG-4标准的基础上进行了开发，不仅显著提高了压缩比，还带来了更高质量的画面效果，目前被广泛应用于高清晰度电视中。另外还有H.265、H.266标准，它们在视频传输和存储方面展现出

了更显著的优势。据业内专家预计，H.266标准可能要到2027年左右才会被广泛接受和使用。

# 任务2 体验数字媒体存储技术

数字媒体存储技术是指用于存储、管理和检索数字媒体内容（如音频、视频、图像等）的技术。随着数字媒体的普及，存储技术在容量、速度、可靠性和成本等方面不断发展。

# 1.数字媒体存储常见设备

数字媒体存储基本设备涵盖多种类型，包括硬盘驱动器（HDD）、固态硬盘（SSD）等内置存储设备；SD卡、USB闪存驱动器、光盘和DVD等便携式存储介质。这些设备提供了大容量、高速度和可靠性的存储解决方案，使用户能够轻松存储、备份和共享各种数字媒体文件。随着技术的不断进步，云存储服务也成为了数字媒体存储的重要选择。

# 2. 数字媒体存储性能指标

# (1) 存储容量

存储容量是指存储介质能够存储的最大数据量，通常以字节（Byte）为单位，常见的容量单位包括 KB、MB、GB、TB 等。这个指标直接决定了用户可以在存储介质上保存多少数字媒体内容。

# (2) 传输速率

传输速率是指存储介质在读取或写入数据时的速度，通常以每秒传输的比特数（bps）或字节数（Bps）为单位。传输速率越快，用户访问和传输数字媒体内容的速度就越快，从而提高了工作效率和用户体验。例如，机械硬盘（HDD）的传输速率一般为  $100\mathrm{MB / s}$  到  $200\mathrm{MB / s}$ ，而固态硬盘（SSD）的传输速率通常能达到  $500\mathrm{MB / s}$  到  $3500\mathrm{MB / s}$  甚至更高。因此，固态硬盘的存储性能远高于机械硬盘，尤其是在随机读写和小文件操作方面，固态硬盘的优势更加明显。

# 3.云存储

随着存储技术的持续进步，云存储作为一种新兴的存储手段，正变得越来越普及。用户能够将数字媒体资源上传到云服务器上进行保存。云服务器，作为服务器的一种类型，不同于传统的物理服务器，它是虚拟化的，并继承了传统物理服务器的所有功能，且功能更为强大。

云存储通过以集群应用、网格技术以及分布式文件系统等功能为技术基础，借助存储虚拟化技术，将网络中大量不同类型的存储设备通过应用软件整合起来，实现协同工作，共同为用户提供数据存储和业务访问服务。例如，百度网盘是云存储的一种典型应用，如图3-17所示。手机中的云盘（或云空间）同样属于云存储的应用范畴，如图3-18所示。

# 1. 什么是数字媒体传输技术

数字媒体传输技术是通过现代通信技术和计算机网络技术，实现数字媒体内容（如图像、声音、视频等）的高效、快速传输。

常见的数字媒体传输技术主要有流媒体技术、P2P传输技术、HTTP/HTTPS传输技术、FTP传输技术和无线传输技术等。

随着短视频平台越来越流行，目前流媒体技术应用非常广泛，它可分为实时流式传输和顺序流式传输两种。实时流式传输注重即时性，适用于在线直播等场景；顺序流式传输则更注重稳定性，适合视频点播等应用。

# 1. 相机的分类

# (1) 单反相机

单反相机即单镜头反光相机，具有一个可更换的镜头，以及一个可动的反光板结构。其原理是光线从镜头进入相机后，被反光板反射到上方的五棱镜，然后折射入取景器，使人眼能够观察到实时画面。按下快门时，反光板抬起，光线直接射到感光元件上曝光成像，如图4-1所示。

# (2) 无反相机

无反相机就是无反光镜的可换镜头相机。与单反相机相比，无反相机取消了单反相机上标志性的的反光板和五棱镜结构，机身更加轻便。光线经过镜头后直接照在感光元件上曝光成像，如图4-2所示。

# 任务2 体验摄影基础操作

摄影基础操作是决定影像质量的技术核心，涵盖对焦机制、曝光控制（光圈/快门/ISO）与色彩管理三大技术要素。通过对这些要素进行规范化的操作和科学的参数配置，可以为影像清晰度、明暗层次与色彩还原提供技术支持。

# 1. 对焦

拍摄的照片与否成功，首先看对焦是否准确、照片关键部分是否清晰。

数码相机一般有两段式快门按钮。自动对焦时一般先半按快门，相机自动对焦系统开始工作，镜头的对焦环自动旋转以完成对焦工作。手动对焦时则需手动

操作镜头对焦环控制焦点位置。

# (1) 自动对焦 (AF)

自动对焦是相机驱动镜头对焦马达，使拍摄主体清晰对焦的功能。自动对焦又分为单次自动对焦模式和连续自动对焦模式。自动对焦的常用场景是快速捕捉动态物体（如运动、野生动物、需抓拍的纪实摄影等）。根据拍摄需求，又可在相机上设置不同对焦模式。

单次自动对焦：半按快门后对焦点锁定在固定位置，适合静物摄影。

连续自动对焦：半按快门后对焦点自动追踪运动物体，适合运动摄影。

# （2）手动对焦（MF）

手动对焦由拍摄者手动旋转镜头对焦环，调整焦点位置以实现清晰成像，适合需要精确对焦或自动对焦难以正常工作的场合。

# 2. 曝光三要素

在数码摄影过程中，光线经由相机镜头投射至图像传感器，传感器上的感光元件接收光信号，并将其转换为电信号或数字信号，进而形成潜在或可见影像，这一过程即为曝光。

# （1）光圈

光圈是位于镜头内部的一种由多个叶片组成的机械装置，通过调整叶片的开合程度，能够精确控制进入相机的光线量。

光圈的作用：一是控制进光量。在光线较暗的环境下，如室内或夜晚，使用大光圈（如f/2.8）可以让更多光线进入相机，使拍摄对象更明亮；在光线充足的环境，如晴天的户外，小光圈（如f/11）能减少进光量，避免画面过亮。二是控制景深。大光圈会使景深变小，即画面中焦点前后的清晰范围变小，背景虚化效果明显，常用于人像摄影、特写摄影等，如图4-13所示；小光圈则使景深变大，画面从前景到背景都能保持相对清晰，适合拍摄风景、建筑等需要大景深的题材，如图4-14所示。

# (2) 快门

快门是相机中用来控制光线照射感光元件时间的装置，是相机的一个重要组成部分。快门开启时间越长，光线进入时间越长，进光量越多；快门开启时间越短，进光量自然越少。

![](images/952913ebe4b3bc61d70d3e11395ce924c8da23190dd07fec6111766e93de74af.jpg)  
图4-13大光圈f/2.8拍摄的照片

![](images/090a70d0343dbd42e06731a714b73bc51377cc37f1c23fe84c5dded38f4f1c7f.jpg)  
图4-14小光圈f/22拍摄的照片

快门的作用：一是控制进光量，快门速度与进光量成反比，较快的快门速

度进光量少，较慢的快门速度进光量多。二是对画面的影响，快速快门（如1/2500s以上）常用于凝固运动物体，让运动中的物体在画面中清晰定格；慢速快门（如1/30s）则会使运动物体产生拖影，形成动感模糊的效果。图4-15所示为高速快门1/2500样片、图4-16所示为低速快门1/30样片。

# （2）感光度

感光度，又称为ISO，用于衡量相机感光元件对光线的敏感程度。

ISO值越高，感光元件对光线越敏感，在相同的光圈和快门速度下，进光量就越多；ISO值越低，感光元件对光线的敏感度越低，进光量越少。

感光度的作用：在光线不足的情况下，提高ISO值可以增加相机的感光度，使照片获得足够的亮度。但需要注意的是，ISO值升高会导致画面噪点增多，画质下降，如图4-17所示。

# 3. 调节色温与白平衡

# （1）色温

色温是描述光源颜色的物理量，单位为开尔文（K），数值高低对应光线的“冷暖”。

低色温（2000K-4000K）：暖色调（如烛光、白炽灯）。

中色温（4000K- 6000K）：中性白（如正午阳光）。

高色温（6000K-10000K）：冷色调（如阴天、荧光灯）。具体色温如图4-18所示。

# （2）白平衡

白平衡是由相机内置程序调整照片色彩，使照片中的白色物体在不同光源下仍呈现真实白色的过程，从而纠正因光源色温差异导致的颜色偏差（如偏黄

或偏蓝），确保物体在不同光线下呈现真实色彩。

# 1.构图

构图是照片画面上的布局、结构。其具体含义是运用相机镜头的成像特征和摄影造型手段，根据主题思想的要求，组成一定的画面，使客观对象比现实生活更富有表现力和艺术感染力，更充分更好地揭示一定的内容。常见的构图方法有中心构图法、三分法构图、对称构图法和引导线构图法。

# 2.景别

景别是指由于相机与被摄体的距离不同，或者镜头焦距的变化，而形成的被摄体在画面中呈现出的范围大小的区别。

根据被摄体在画面中呈现的范围，景别通常分为以下几类，如图4-23所示。

# （1）远景

远景是展示广阔空间和宏大场面的景别，常用来表现自然风景、城市全貌、大型活动现场等。其画面涵盖范围广，着重突出环境的整体氛围和气势，人物在画面中所占比例较小。

# （2）全景

全景能够完整地表现人物的全身或场景的全貌，在展示人物的同时，也能呈现出人物与周围环境的关系。

# （3）中景

中景呈现人物膝盖以上或物体的主要部分，此时人物的动作和表情成为画面的重点，观众可以清晰看到人物之间的交流和互动。

# （4）近景

近景主要表现人物胸部以上或物体的局部，更加突出人物的面部表情和情感变化，以及物体的细节特征。

# （5）特写

它通常只展示人物肩部以上的部分，甚至可能更加聚焦在特定的细节上，如眼睛、手部等。特写通过极度放大的细节来强调某一特定的元素，从而产生强烈的视觉冲击力和情感震撼。

# 中国摄影发展史：光影记录的时代变迁

中国摄影发展史是一部用光影书写的视觉史诗，记录着社会变迁与文化演进。从早期西方传入到当代蓬勃发展，中国摄影走过了独特的发展道路。

# 1.萌芽期（1840年-1911年）

摄影术随西方列强传入中国，最初用于商业人像拍摄和地理勘察。香港、上海等地出现首批照相馆，拍摄内容以肖像、风景为主。这一时期摄影技术主要掌握在外国人手中，但已开始有中国人学习并开设照相馆。

# 2.探索期（1912年-1949年）

摄影开始应用于新闻报道和艺术创作。郎静山等摄影家尝试将中国传统美学与摄影结合，开创“集锦摄影”风格。抗日战争时期，摄影成为重要的宣传工具，沙飞等战地摄影师记录下珍贵的历史画面。

# 3. 发展期（1950年-1978年）

新中国成立后，摄影成为重要的宣传媒介。《人民画报》等刊物大量使用摄影图片。这一时期的摄影作品以社会主义建设为主题，强调集体主义和革命精神，形成了独特的视觉风格。

# 4. 转型期（1979年-1999年）

改革开放带来思想解放，摄影创作呈现多元化趋势。“四月影会”等摄影团体推动艺术摄影发展。纪实摄影兴起，侯登科、解海龙等摄影师关注社会现实，创作出具有人文关怀的作品。

# 5.繁荣期（2000年至今）

数码技术革命彻底改变了摄影生态。摄影创作群体扩大，题材更加丰富。当代摄影艺术家如王庆松、洪浩等，运用摄影探讨社会议题，作品在国际上获得认可。手机摄影和社交媒体的普及，使摄影成为大众日常生活的一部分。

中国摄影发展史折射出社会文化的变迁，从最初的模仿学习到形成独特风格，再到融入当代艺术语境，展现出强大的生命力和创造力。未来，随着技术进步和文化交流的深入，中国摄影必将继续创新发展，在世界摄影艺术舞台上发挥更大作用。

# 任务 初探摄像拍摄技巧

固定镜头与运动镜头是镜头语言的基本单位，通过稳定与变化的空间感实现动态影像的叙事功能，承担着为动态影像传递信息和情绪的任务。

# 1. 固定镜头

固定镜头是指摄像机在拍摄过程中位置固定，镜头角度和焦距不变的拍摄方式。固定镜头通常用于表现静态场景或强调某一特定对象，能够给观众带来稳定、安静的视觉感受。固定镜头的特点是画面稳定，适合用于对话场景、静态展示等。

# 2. 运动镜头

运动镜头是指摄像机在拍摄过程中位置、角度或焦距发生变化的拍摄方式。运动镜头可以通过推、拉、摇、移、跟、升降等手法，增强画面的动态感和视觉冲击力。运动镜头常用于表现动态场景、追踪运动对象或引导观众的视线。

# 中国摄像技术的百年奋斗与创新之路

中国摄像技术的发展历程，是一部从无到有、从模仿到创新、从追赶到引领的奋斗史。它见证了中国科技的进步，也折射出中华民族自强不息的精神。

# 1.萌芽与探索（20世纪初-1949年）

20世纪初，摄像技术作为“舶来品”传入中国。最初，主要用于电影拍摄，出现了《定军山》等一批早期电影作品。这一时期，中国摄像技术处于萌芽阶段，主要依赖进口设备和技术，国内尚未形成自主研发能力。

# 2. 起步与发展（1949年-1978年）

新中国成立后，国家高度重视科技发展，摄像技术也迎来了新的发展机遇。1958年，北京电影制片厂成功研制出中国第一台35毫米电影摄影机“红旗牌”，标志着中国摄像技术迈出了自主创新的第一步。

# 3.改革与腾飞（1978年-2012年）

改革开放为中国摄像技术发展注入了强劲动力。随着市场经济的发展，摄像技术应用领域不断拓展，从电影、电视延伸到教育、医疗、安防等各个领域。与此同时，中国摄像企业如雨后春笋般涌现，积极参与国际竞争，涌现出大疆、海康威视等一批优秀企业。

# 4. 创新与引领（2012年至今）

进入新时代，人工智能、大数据、云计算等新一代信息技术的快速发展，为摄像技术带来了新的机遇和挑战。中国摄像企业紧跟时代潮流，在超高清、智能感知、虚拟现实等领域取得了一系列突破性进展，部分技术已达到国际领先水平。

未来，中国摄像技术将继续朝着超高清、智能化、网络化、融合化的方向发展。随着5G、人工智能等技术的不断成熟，摄像技术将在更多领域发挥重要作用，为经济社会发展做出更大贡献。

# 2. 图形图像处理常用技术

# （1）数字图像处理技术

数字图像处理的关键技术包括了图像增强、图像复原、图像分割、特征提取和图像压缩等技术。

图像增强是一种通过算法处理优化图像信息表达的技术手段，旨在改善图像的视觉质量或突出图像中的某些特征，以便更好地进行分析或展示。图像增强的目标不是恢复原始图像，而是通过处理使图像更适合特定应用。

图像复原是指通过各种技术手段修复受损或退化的图像，使其尽可能恢复到原始状态。常见的复原技术包括去噪、去模糊、修复缺失区域、增强分辨率等。

图像复原和图像增强是图像处理中的两个重要领域，尽管它们的目标都是改变图像质量，但是它们的方法、目的和应用场景存在显著区别，具体参见表5-1。

图像分割就是把图像分成若干个特定的、具有独特性质的区域，并提取其中感兴趣部分的技术和过程。一般来说分割的这些区域互不相交，它们在颜色、亮度或纹理等方面与临近图像有着差别，每个区域对应实际场景中的某个物体或某个特定部分，人们可以对每个区域进行单独的处理和分析。

图像分割在图像处理中发挥着重要作用，是后续图像分析、图像识别的前提和基础。例如自动驾驶汽车需要实时识别道路、车辆、行人等障碍物，并根据识别结果进行路径规划和避障操作。通过图像分割技术可以将图像中的不同对象分割开来，从而为自动驾驶汽车提供更准确的环境感知信息。

特征提取是从图像中提取出有意义的、可区分的特征，以便于分类、识别或其他任务，如人脸识别、指纹识别等。图像分割和特征提取在图像处理中是相辅相成的。图像分割侧重于将图像划分为多少个区域或对象，目的是分离图像中的不同部分。特征提取侧重于从图像中提取有意义的特征，目的是描述图像的内容。图像分割可以为特征提取提供更精确的区域，使得特征提取可以在特定的区域内进行，从而提高特征的准确性和代表性。特征提取的结果可以用于图像分割，例如通过提取的特征来判断像素或区域是否属于同一对象。

由于图像数据之间存在一定的冗余，包括空间冗余、时间冗余、视觉冗余和编码冗余等。图像压缩技术可以减少图像数据的冗余，同时尽可能保持图像的视觉质量，便于存储和传输。

# （2）计算机图形处理技术

计算机图形处理技术主要围绕图形生成与建模、图形渲染和人机交互等几个方面。

图形生成与建模是计算机图形学中的核心研究内容之一，主要包括基本图形生成、曲线与曲面建模、三维建模和自然现象建模等。

基本图形生成是计算机图形学的基础，研究如何生成简单的几何图形，例如点、线、圆、多边形等。图5-1所示为利用简单几何图形生成的笔记本图形案例。

曲线与曲面建模是图形生成的高级内容，研究如何用数学方法描述复杂的曲线和曲面，例如利用控制点定义平滑曲线的贝塞尔曲线和利用控制网格定义平滑曲面的贝塞尔曲面。

人机交互是指用户通过输入设备与计算机进行交互，计算机通过输出设备向用户反馈信息的过程。在计算机图形学中，人机交互的核心目标是设计直观、易

# 任务1 认识数字视音频技术

数字视音频技术改变了人们观看和聆听世界的方式，带来更加沉浸和真实的视听体验。

# 1. 什么是数字视音频技术

数字音频、数字视频与后期特效是数字媒体技术的主要应用领域之一。

数字音频技术是利用数字化手段录制、存放、编辑、压缩和播放音频的技术。

数字视频技术是利用数字化手段进行视频存储、编辑、压缩和播放的技术。

影视特效是对现实生活中不可能完成以及难以完成的镜头，或需要花费大量资金拍摄的镜头，用计算机或工作站进行数字化处理，从而达到预期的视觉效果。

数字特效是利用计算机技术创建、修改或增强影像中的视觉效果，广泛应用于电影、电视、游戏等领域。

# 2. 数字视音频关键技术

# （1）数字音频处理技术

语音合成又称文语转换，它是将计算机自己产生的或外部输入的文字信息转变为拟人化的、高自然度的语音的输出技术。传统的语音合成方法包括拼接合成和参数合成，前者通过拼接预录制的语音片段实现，后者则依赖数学模型生成语音。近年来，深度学习技术的引入极大地推动了语音合成的发展，语音合成的自然度和准确度不断提升，已经能够生成更加自然、流畅的语音。

语音识别是将语音信号转化为可理解文本形式的技术。它通过分析语音的声学特征，识别出语音中的音素、词汇和语句，并将其转化为可读的文本形式。随着深度学习的兴起，基于深度神经网络的语音识别技术取得重大突破，大幅降低了识别错误率。近年来，端到端的语音识别模型的出现，使其能够在复杂环境下实现更高效、精准的语音到文本转换。

# （2）数字视频处理和后期特效处理技术

数字视频编辑采用的是非线性编辑方法，这是一种基于数字化的视频编辑方法，允许用户自由访问、调整和重组视频片段，无需按时间顺序操作。与传统的线性编辑相比，非线性编辑制作更灵活高效，编辑造成的视音频质量损失几乎可以忽略不计，非线性编辑系统设备更小型、功能集成度更高、价格更低。典型的非线性编辑过程为：将素材导入编辑软件，利用软件提供的功能进行剪辑、添加转场、添加特效等，最后进行渲染导出。

抠像技术能将视频中的主体（如人物）从原背景中精准分离，便于后期自由替换场景或叠加特效。传统方法依赖绿幕或蓝幕拍摄，通过颜色差异识别主体，随着技术突破限制，无需绿幕也能识别边缘。抠像技术在电影特效中广泛应用，

演员在绿幕前表演，后期替换为太空或战场等场景；天气预报中，主持人站在绿幕前，播出时背景替换为动态地图；短视频创作中，用户可通过手机APP一键更换背景。

调色技术主要是通过调整画面的色调、对比度、饱和度等参数，来增强视觉效果，营造特定氛围或风格。它不仅能够修正拍摄时的光线、色彩偏差，还能根据创作需求进行艺术化的色彩重塑，使作品更加符合导演的创意意图，提升整体的视觉吸引力和情感表达。

# 任务2 体验数字视音频技术

数字视音频技术涉及多种软件工具，涵盖编辑、处理、编码、流媒体等不同用途。

# 1.音频制作软件

在数字音频创作领域中，常见的软件有Adobe Audition和GoldWave。

Adobe Audition 是 Adobe 公司开发的专业音频编辑软件，支持多音轨、多种音频格式、多种音频特效，可以进行录音、剪辑、效果处理等操作，它广泛应用于音乐制作、影视音频处理、广播电台节目及广告制作、游戏音效设计等多个领域。

GoldWave 是一款经典且广受欢迎的音频编辑软件，集声音编辑、播放、录制和转换等功能，主要应用于个人创作、教育领域音频处理、中小型商业项目制作及基础音频修复等场景，其直观界面和基础功能集使其成为初学者及轻量级用户的理想选择。

数字视频编辑软件又被称为非线性编辑软件，主要用于对数字视频信号进行编辑处理。常见的数字视频编辑软件有 Adobe Premiere、剪映等。

# 任务1 认识数字动画技术

动画是让静止的画面跃动成诗的艺术形式，它广泛应用于影视、游戏、广告等领域，既是表达创意的媒介，也是传递情感与故事的文化载体。

# 1. 什么是动画

动画是一种通过连续播放静态图像或模型，利用人眼的视觉暂留现象产生动态效果的艺术形式与技术。其核心在于通过逐帧的变化来模拟运动或变化的过程，从而让观众感知到连贯的动作或叙事。

# 2. 动画原理

视觉暂留是指人眼在观察景物时，光信号传入大脑神经需要经过一段短暂的时间。当光的作用结束后，视觉形象并不立即消失，而是会在视网膜上停留0.1至0.4秒的时间，这种现象被称为“视觉暂留”。

在日常生活中，静止的风扇在快速旋转时，几片扇叶会连成一个圆盘；燃放的冷光烟花在空中晃动，肉眼看到火花因为运动形成连续曲线或形状，如图 5-26 所示。

这些现象都是因为人眼的视觉暂留特性。也就是说当人们观察的一个影像消失后，如果在很短的时间内，紧接着第二个相关的影像马上进入了人眼的视网膜，这时就会给人眼造成连续影像动起来的错觉。电影正是利用这一原理将连续的图

像以24帧/秒或更高速度播放时，残留影像重叠形成流畅运动幻觉。

似动现象是一种通过视觉幻觉产生的运动假象，观察者在静止的物体间看到了运动，或在没有连续位移的地方看到了连续运动。这种现象的产生是由于视觉系统的特殊处理方式：当两个刺激物按一定的空间和时间间隔相继呈现时，观察者会感觉到一个刺激物在向另一个刺激物做连续运动，但实际上物体并不是在动，而是“感觉上在动”，这个现象被称为“似动现象”。如图5-27所示。

# 3. 动画的分类

动画的分类纷繁复杂，最常见的有二维动画、三维动画和定格动画，其中使用了计算机技术的二维动画和三维动画又可统称为数字动画。

二维动画：也称为2D动画，是一种传统的动画形式，它通过在二维平面上绘制和操纵图像来创造运动幻觉。可以通过手工绘制或使用计算机软件制作。

三维动画：也称为3D动画，是通过计算机图形技术创建的动态影像。在三维软件中，设计师通过建模、绑定、动画、灯光和渲染等步骤，将三维模型转化为逼真的动态图像。这些图像通过快速显示，从而创造出动态的效果。

定格动画：是通过逐格地拍摄对象使之连续放映，从而产生仿佛活了一般的人物或任何想象中的角色。通常所指的定格动画一般都是由黏土偶，木偶或混合材料的角色来演出的。这种动画形式的历史和传统意义上的手绘动画历史一样长，甚至可能更古老。

# 4. 数字动画的关键技术

数字动画是利用计算机软件和数字技术进行动画创作与制作的现代形式。相比传统手绘，它大幅提升了效率、降低了成本，并扩展了艺术表现的可能性。数字动画的关键技术主要有关键帧、动作捕捉和物理模拟技术等。

# （1）关键帧技术

关键帧是动画制作中用于定义运动起止点或转折点的核心画面。其本质是通过“选择性定义”动作的极端状态，动画师绘制或由计算机自动生成中间过渡帧，从而高效构建连续动作序列。这一概念源于传统手绘动画的“原画-中间画”分工模式，在数字时代演变为非线性编辑的核心技术之一。

# (2) 动作捕捉技术

动作捕捉技术是通过在生物体关键解剖位点部署传感器，实时记录运动轨迹并映射至数字模型的核心技术。系统通过反光标记或惯性测量单元采集肢体空间坐标数据，这些数据经计算机解算后驱动虚拟角色运动。该技术突破了传统关键帧动画的局限性，尤其在面部微表情和手指关节运动等生物力学特征的数字化重建方面具有显著优势。如图5-28所示为身体动态捕捉。如图5-29所示为面部表情捕捉系统。

# （3）物理模拟技术

物理模拟技术通过计算机程序模仿真实世界的物理规律，让虚拟物体像现实中一样运动变化。它能精确计算物体移动、变形和碰撞的过程，比如模拟布料自然下垂的褶皱、水流遇到障碍物的波动，或是爆炸时碎片飞溅的轨迹。在动画制作中，这项技术让角色的衣服随风摆动更逼真，使水流冲击岩石溅起的水花更自然，还能准确呈现爆炸场景中碎片飞散的物理逻辑。通过科学算法与艺术设计的结合，虚拟世界中的物理现象既能符合科学规律，又能满足视觉表现需求。

# 1. 二维动画制作软件

二维动画制作软件主要有 AdobeAnimate 和 TVPaint Animation 等。

AdobeAnimate利用矢量绘图引擎和骨骼绑定系统，可快速实现补间动画与交互式内容创作，尤其适合网络动画及广告短片制作。它与Adobe生态无缝衔接的特性大幅提升了跨软件协作的流畅度。

TVPaint Animation 广泛用于传统手绘动画、定格动画和数字动画创作，其仿纸面绘画的光桌模式和多平面摄影机功能，为艺术动画短片提供了从线稿扫描到动态上色的全流程支持。

# 2. 三维动画制作软件

二维动画制作软件主要有 3ds Max、Maya、和 Blender 等。

3ds Max 在建筑可视化与游戏过场动画领域表现卓越, 其内建的 Particle Flow 粒子系统和 Forest Pack 植被生成器, 可高效完成大规模场景的动态模拟。

Maya 作为电影工业的标准工具，凭借 Bifrost 流体模拟系统和 HumanIK 高级角色绑定模块，成为复杂角色动画与特效制作的首选。

Blender 作为开源全流程工具提供了从建模、雕刻到渲染的一站式解决方案，其创新的 Grease Pencil 模块甚至支持在 3D 空间中直接绘制 2D 动画，实现跨维度的艺术表达。

# 3. 人工智能辅助动画制作工具

人工智能辅助动画制作工具通过自动化与智能化技术显著提升了动画制作的效率与质量。它们能够自动生成中间帧、优化角色动作、简化场景构建，并实时渲染，减少人工干预。AI还能分析大量数据，提供风格迁移、面部表情捕捉和语音同步等功能，使角色更加生动。例如Adobe Character Animator软件可以通过摄像头和麦克风实时捕捉面部表情和语音，自动生成角色动画，简化了角色表演的制作流程；DeepMotion工具利用AI和机器学习技术，将视频中的动作转化为3D动画，适用于角色动作捕捉和运动生成。

# 制作《小球形变弹跳》二维动画

制作一段时长约1-3秒的二维动画，生动形象地展示小球从高处落下、与地面碰撞发生形变、并再次弹起的运动过程。要求动画流畅自然，能够清晰体现小球的形变及运动轨迹的变化。

# 1. 任务分析

# （1）策划阶段

理解主题：《小球形变弹跳》二维动画制作任务，要求展示小球垂直弹跳的物理运动规律。主要运动过程包含“下落加速  $\rightarrow$  触地挤压形变  $\rightarrow$  弹起恢复形变  $\rightarrow$  运动轨迹循环”。

利用形变（压扁小球）和缓动效果模拟真实弹跳；时间轴规划总时长为30帧；关键动作帧分别为：第1帧为起始位置（顶部）  $\rightarrow$  第10帧为触地（底部） $\rightarrow$  第13帧为最大形变（压扁）  $\rightarrow$  第25帧为弹起至顶点  $\rightarrow$  第30帧为延长帧（循

环衔接）。

经过分析，采用二维动画制作软件AdobeAnimate2021来进行制作，关键步骤包括：创建舞台、绘制元件、关键帧设置、位置调整、形变操作、补间动画、调整补间属性。

# （4）测试评估阶段

根据输出的效果，进行细节改动。

# （5）反馈改进阶段

以小组为单位，对完成的小球形变弹跳动画效果进行交流、分析、评价，可以从动画流畅度、物理效果、形状变化等方面开展。

# 中国优秀动画电影作品：《哪吒之魔童闹海》

《哪吒之魔童闹海》作为一部备受瞩目的动画电影，其动画技术在多个方面展现出了卓越的成就与独特的魅力。

# 1. 画面表现力

场景设计：影片的场景设计丰富多样且极具想象力，从陈塘关的古朴风貌到海底世界的奇幻绚丽，再到玉虚宫的庄严肃穆，每一个场景都精心打造，细节丰富。例如，陈塘关的建筑风格、街道布局等都充满了历史感和生活气息，而海底世界的各种奇异生物、珊瑚礁等则展现了神秘莫测的水下景观。

角色建模：角色建模精细，人物形象生动逼真。哪吒的顽皮与坚毅、敖丙的沉稳与内敛、太乙真人的仙风道骨等，都通过细腻的建模和表情捕捉得以展现。特别是哪吒的表情变化丰富，无论是愤怒、惊讶还是搞怪，都能让观众感受到他的情感起伏。

# 2.特效制作

粒子特效：在特效方面，影片运用了大量的粒子特效来增强画面的视觉冲击力。比如，哪吒与敖丙大战时的技能释放，火焰、水柱、闪电等特效元素交织在一起，形成了绚丽多彩的战斗画面。这些粒子特效不仅效果逼真，而且与角色的动作和场景完美融合，为观众带来了震撼的视觉体验。

流体模拟：对于流体的模拟也是影片的一大亮点。海水、岩浆等流体物质的运动自然流畅，如“水漫陈塘关”场景中，海水的涌动、翻滚和冲击，给观众带来了强烈的视觉冲击和身临其境的感觉。

# 3.技术创新

动态水墨渲染引擎：制作团队自主研发的“动态水墨渲染引擎”，将中国传统水墨画的“气韵生动”融入3D动画，实现了毛笔在宣纸上晕染渗透的实时模拟。这一技术不仅破解了传统水墨数字化的难题，还构建起了东方美学的视觉语法体系，为影片增添了独特的艺术风格。

AI算法的应用：AI算法在影片的制作中发挥了重要作用。它学习中国传统水墨画风格，自动生成背景纹理；预测水流、火焰的物理轨迹，减少手动调参的繁琐；实时捕捉演员表情，让角色的表情更加生动自然。

# 3.文化表达

传统文化元素的融入：影片巧妙地将中国传统文化元素融入到动画技术中，如敦煌壁画色彩、山海经神兽造型等，通过纹样生成、动作库匹配等方式，将传统武术动作融入哪吒打斗场景，既展现了传统文化的魅力，又增强了影片的文化内涵。

主题的深化：延续了“我命由我不由天”的主题，并通过技术手段强化了这一表达。哪吒重铸肉身的场景通过七色宝莲的特效设计，传递了生命重生的哲学意义，让观众在欣赏精彩画面的同时，也能感受到深刻的文化内涵。

# 任务1 认识虚拟现实技术

虚拟现实技术自20世纪中叶诞生以来，经历了漫长的发展历程。从最初的概念萌芽到如今的实际应用，虚拟现实技术在多个领域产生了深远的影响。

# 1. 什么是虚拟现实

虚拟现实（Virtual Reality）是一种通过计算机技术模拟生成三维虚拟环境，并利用传感器、显示设备等硬件使用户能够沉浸其中并与之交互。其核心目标是通过高度仿真的视听、触觉甚至嗅觉反馈，让用户产生身临其境的感觉。

虚拟现实技术是人类和计算机之间进行复杂数据交互的技术，它创造一种人工仿真环境，向用户提供视觉、听觉、触觉、味觉和嗅觉等感知功能，人们能够在这个虚拟环境中观察、聆听、触摸、漫游，并能与虚拟场景中的物体进行互动。

# 2.虚拟现实的特征

虚拟现实技术具有三个主要特征，分别是沉浸感、交互性和想象性。

# （1）沉浸性

沉浸感又称临场感，是用户在虚拟环境中的主观感受，即用户在虚拟世界中所感受到的真实程度。理想情况下，这种虚拟环境应使用户难以分辨真假，全身心地投入到计算机创建的三维立体环境中。沉浸感不仅包括视觉和听觉的逼真体验，还涉及触觉、嗅觉等多维感官的模拟。例如，用户在虚拟环境中移动时，视场中的物体应根据用户的移动而移动，同时用户也能听到三维空间的声音，甚至通过特殊的设备感受到虚拟物体的形状、重量等。

# （2）交互性

交互性是虚拟现实技术的关键特征之一，它允许用户与虚拟环境进行实时互动。用户可以通过各种输入设备对虚拟环境中的对象进行操作，并立即得到相应的反馈。这种交互不仅限于简单的点击或触摸，还可以包括复杂的动作和手势识别。例如，用户可以直接用手抓取虚拟环境中的物体，感受到物体的重量和质地，并与之进行互动，从而增强参与感和真实感。

# （3）想象性

想象性也称为构想性或想象力，是设计者在创建虚拟环境时发挥创造力的空间。虚拟环境是由人类想象和创造出来的，它可以用于实现各种目标和场景。在虚拟现实系统中，设计者可以根据需要自由地构建和修改虚拟世界，创造出超越现实的奇幻场景或模拟现实世界的各种情境。这种想象性为用户提供了无限的探索空间和创意可能性。

# 3.虚拟现实的系统组成

一个典型的虚拟现实系统主要由计算机、输入输出设备、应用软件和数据库等部分组成。

# （1）计算机

在虚拟现实技术中，计算机扮演着不可或缺的核心角色，它是整个虚拟环境的“中枢系统”。作为系统运行的关键，其主要承担着虚拟场景的即时渲染处理、人机交互数据的实时运算等核心任务。由于虚拟世界构建具有极高的复杂度，特别是在处理大规模精细场景时，所需的运算资源呈几何级数增长，这就对计算机硬件性能提出了非常高的标准和要求。

# (2) 输入输出设备

虚拟现实系统要求用户采用自然的方式与虚拟世界进行交互，传统的鼠标和键盘是基本无法实现这个功能的，这就需要采用特殊的交互设备，用来识别用户各种形式的输入，并实时生成反馈信息，将反馈信息输入虚拟环境中。目前，常见的虚拟现实交互设备主要有 VR 手柄和数据手套等，如图 6-1、6-2 所示。

# （3）应用软件

在构建虚拟现实系统的过程中，需要多种专业软件的协同配合。这些软件在创建虚拟环境所需素材方面发挥着重要作用。例如，设计人员通常会先运用设计软件对原始采集的数据和图像进行预处理；再利用三维建模软件对模型进行纹理贴图等操作；同时借助音视频编辑软件制作所需的音视频素材。

为了将这些分散的多媒体元素有机整合，构建出具有交互功能的完整虚拟空间，必须使用专业的虚拟现实开发引擎。这类引擎主要负责虚拟现实系统中的多个核心功能，包括三维模型的整合、场景渲染、动画脚本控制以及音视频的同步呈现等关键任务。

# （4）数据库

在虚拟现实系统中，数据库承担着海量数据存储与管理的重要职能。它系统性地保存着构建虚拟环境所需的各类基础数据，涵盖地理空间信息、三维场景构建、动态模型等多个维度的数据资源。具体而言，虚拟场景中呈现的每一个实体对象，都需要在数据库中建立对应的数字化模型。

# 任务1 认识增强现实技术

增强现实技术起源于20世纪60年代，经历了从实验室研究、技术突破到商业化应用的发展历程，如今已在多个领域广泛应用并持续进步。

# 1. 什么是增强现实

增强现实（Augmented Reality，简称 AR）是一种融合了计算机视觉、传感器技术、人工智能和 3D 渲染等多种技术的交互式数字体验。它通过摄像头、智能设备或 AR 眼镜采集现实世界的信息，并将计算机生成的虚拟内容（如图像、声音、文本和 3D 模型等）叠加到真实环境中，使用户能够在现实世界中感知并与虚拟元素交互，从而增强对现实的理解和体验。

与完全沉浸于虚拟环境中的虚拟现实不同，增强现实不会取代现实世界，而是对现实世界进行增强，使虚拟内容与真实世界无缝融合。通过实时感知、空间定位和交互操作。

# 2.增强现实的主要特征

增强现实技术融合了虚拟世界与现实环境，使用户能够在真实世界中感知并

与虚拟信息互动。其主要特点包括真实与虚拟融合、实时交互和空间感知能力。

# (1) 虚实结合

虚实结合是增强现实技术的核心特征，它指的是虚拟物体和现实世界无缝融合，使用户在现实环境中能够看到并与虚拟内容进行互动。与虚拟现实完全沉浸在虚拟世界不同，AR通过在现实环境中叠加虚拟物体、数据和信息，使用户能够同时看到真实世界与虚拟内容。这种虚实结合让用户可以在不离开现实环境的情况下，与虚拟物体进行互动，从而拓展了感知和操作方式。例如，通过手机或AR眼镜观看现实世界时，虚拟信息（如家具、指示标志等）可以被准确地叠加在实际环境中，从而提供增强的视觉感知和交互体验。

# （2）实时交互

实时交互是增强现实技术的另一大特征，意味着用户与虚拟物体之间的互动是即时且响应迅速的。当用户改变自己的位置、视角或进行特定动作时，系统会立刻对这些变化做出反应，并实时调整虚拟内容。实时交互增强了AR体验的沉浸感，因为虚拟物体不会固定不变，而是能够根据用户的实际动作和输入即时调整。例如，用户可以用手指在屏幕上点击、拖动或缩放虚拟物体，系统会立即反馈这些操作，使得用户感受到更加自然、流畅的交互体验。这种特性在AR游戏、教育培训、设计和医疗等领域的应用中尤为重要。

# （3）三维注册

三维注册是增强现实中用于将虚拟物体准确定位到现实世界中的技术。它通过精确的空间对齐和匹配，使得虚拟内容能够正确地叠加在现实场景的对应位置。在AR中，三维注册通常依赖于相机、深度传感器和惯性测量单元等设备获取环境的空间信息，然后通过计算机视觉算法将虚拟对象与这些数据进行匹配，从而确保虚拟物体的空间位置、角度和比例与现实世界一致。这项技术不仅保证了虚拟物体在现实世界中的正确显示，还支持虚拟物体与环境中其他物体的交互。例如，在虚拟家具展示应用中，三维注册能够确保家具的大小、形状、位置与现实房间的空间一致，从而给用户提供更加真实和直观的使用体验。

# 3.增强现实的系统组成

一个完整的增强现实系统涉及多个硬件和软件组件的协同工作。与虚拟现实系统类似，增强现实系统的构成包括计算设备、输入输出设备、相关软件以及数

据库等基本组成部分。与虚拟现实系统相比，增强现实系统中尤其需要强调的是环境感知硬件的作用，如摄像头、深度传感器、位置追踪传感器等。

# （1）摄像头

摄像头是AR系统的核心感知设备之一，主要用于捕捉现实世界的图像和视频，为虚拟内容的叠加提供参考。通过计算机视觉技术，摄像头可以识别物体、追踪手势、检测面部表情，并与虚拟元素进行实时交互。

# (2) 深度传感器

深度传感器用于测量场景中的空间深度信息，使AR设备能够理解物体之间的距离和环境结构。

# （3）位置追踪传感器

位置追踪传感器负责获取设备在空间中的位置信息，确保虚拟对象能稳定地贴合现实世界。

# 4.增强现实与全息投影

增强现实和全息投影技术都致力于提供虚拟与现实世界之间的互动体验，但它们在实现方式、应用场景和技术表现上有所不同。增强现实通过将虚拟信息叠加到现实世界中，为用户提供更多的实时数据和交互内容；全息投影通过光学原理生成真实的三维影像，用户无需佩戴任何设备就可以从不同角度观察这些影像。

全息投影技术与增强现实的结合，能够打破设备屏幕的限制，实现更自然、更真实的虚拟与现实融合。例如，在医疗、教育、商业展示等领域，全息投影可以将虚拟的三维模型直接投射到现实空间中，用户无需依赖特殊设备就能进行直观交互。全息投影能够提供比传统增强现实更强的沉浸感和交互性，使用户体验更为生动、直观。
