82. 答：（本小题4分） 一级缓存位于CPU内部(或是CPU内核的一部分)(1分)，负责在CPU内部的寄存器与外部Cache之间的缓冲(1分)； 二级缓存位于CPU外部(或独立于CPU)(1分)，负责整个CPU与内存之间的缓冲(或弥补CPU内部Cache的容量过小)(1分)。 注意：CPU写成中央处理器；Cache写成缓存、高速缓冲存储器，都是正确的。

83. 答：（本小题4分） 主分区（1分）；扩展分区（1分）；逻辑分区（1分）；非DOS分区（1分）。

84. 答：（本小题5分） TCP：传输控制协议(1分)；IP：网际协议(1分)；POP3：邮局协议(1分)；SMTP：简单邮件传输协议(1分)；UDP：用户数据包协议(1分)。

85. 答：（本小题5分） CSS 样式的中文全称是层叠样式表(1 分)。 外部CSS样式表存储在一个独立的外部CSS(.css)文件中(1分)，可以被不同的网页调用(1分)； 内部CSS样式表存储在HTML文档文件头部分(或HTML文档  $\langle$  style  $\rangle$  ……</style>标签内）（1分），只应用于当前网页中(1分)。
