# 四、填空题：本大题共10小题，每小题2分，共20分。下列每小题的答案填在答题卡对应的横线上。

71. 传输速率为  $72\mathrm{b / s}$ ，意味着每分钟最多传送 ______ 个 ASCII 字符。

答案：540

解题关键词：速率、每分钟

72.分辨率决定了图像所包含像素的最大数目，____决定了图像中不同颜色的最大数目。

答案：色彩位数/图像深度

解题关键词：颜色最大数目

73.已知字母a的ASCII码为八进制数141Q，那么ASCII码为十六进制数65H的字是

答案：e

解题关键词: 熟悉的同学可以知道a十六进制是多少, 同时也可以通过八进制算出来

74.计算机中浮点数由指数和尾数两部分组成，其中 用整数的补码形式表示

答案：指数

解题关键词：整数的补码形式

75. 数据传输中规定时间内出错数据占被传输数据总数的比例称为

答案：误码率

解题关键词：传输错误数据的比例

76.计算机网络中，组成IPv4地址的字节数是

答案：4

解题关键词：IPv4、地址

77. TCP/IP 协议中, DNS 的功能是把 ______ 转换为 IP 地址。

答案：域名

解题关键词：DNS功能

78. 一种支持数字设备之间短距离、低速率、低成本的灵活、安全、小功率的无线通信技术。通信时规定一个为主设备，另一个为从设备，并且由主设备进行查找并发起配对的物联网关键技术是____。

答案：蓝牙技术

解题关键词：主设备进行查找并发起配对

79. 在移动互联网中, 将应用程序的多种服务即不同的功能单元, 通过这些服务之间定义好的接口和契约连接起来、协同工作的组件模型是

答案：面向服务架构/SOA

解题关键词：移动互联网、将应用程序的多种服务即不同的功能单元

80. 在云计算中, 通过网络提供软件的模式, 用户无须购买软件, 而是向提供商租用基于 Web 的软件来管理企业经营活动, 称为

答案：SAAS/软件即服务

解题关键词：云计算、租用软件
