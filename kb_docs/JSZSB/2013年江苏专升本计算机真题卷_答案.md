# 四、填空题

81. 十进制数转换为八进制数时由于整数和小数的转换方法不同，所以先将十进制数的整数部分和小数部分分别转换后，再加以合并。其中，整数部分转换采用“除基取余”法，小数部分转换采用“乘基取整”法。本题中，整数部分十进制数215转换为八进制数327，小数部分0.25转换为八进制数0.2，合并后的结果为327.2。因此，本题填327.2。  
82.“摩尔(Moore)定律”指出单块集成电路的集成度平均每18~24个月翻一番。因此，本题填摩尔定律。  
83. SATA 接口近几年来流行，是当前 PC 机硬盘和主机连接最常使用的接口，它是串行 ATA 接口，以高速串行的方式传输数据，其传输速率高达 150 MB/s(SATA 1.0)、300 MB/s(SATA 2.0) 或 600MB/s(SATA3.0)。由于是串行传输，线缆数目大大缩减，有利于机箱内散热。因此，本题填硬盘。  
84.如果CPU的地址线宽为n，那么CPU的寻址大小是  $2^{\wedge}n(B)$  .本题中，寻址大小是32GB，通过公式倒推，有  $2^{\wedge}n(B) = 32GB = 32X2^{*}30(B) = 2^{*}5X2^{*}30(B) = 2^{*}35(B)$  ，可以得到地线有35根。因此，本题填35。  
85.数码相机镜头和快门与传统相机基本相同，不同之处是它不使用光敏卤化银胶片成像，而是将影像聚焦在成像芯片(CCD或CMOS)上，并由成像芯片转换成电信号，再经模数转换(A/D转换)变成数字图像，经过必要的图像处理和数据压缩之后，存储在相机内部的存储器中。其中成像芯片是数码相机的核心。因此，本题填CCD。  
86. 操作系统的存储管理功能管理内存资源的高效、合理使用。现在，操作系统一般都采用虚拟存储技术进行存储管理。虚拟存储技术是把内存和外存(硬盘)结合起来管理，为用户提供-个容量比实际内存大得多的“虚拟存储器”。因此本题填虚拟存储器。  
87. 关于算法，需要考虑以下三个方面的问题，即如何确定算法(算法设计)、如何表示算法(算法表示)以及如何使算法更有效(算法的复杂性分析)。因此，本题填算法分析。  
88.本题属于基本常识，填验证码。  
89.“蓝牙”是构建无线局域网的一种技术，它是一种短距离、低速率、低成本的无线通信技术，是个人无线局域网协议(802.15)的基础。因此，本题填蓝牙。  
90.本题属于基本概念，填分层。  
91. 输入字符的方法有两类:人工输入和自动识别输入。因此,本题填自动识别。  
92.本题属于基本概念，填流媒体。  
93. 设有关系 R 和关系 S，它们具有相同的模式，其并操作表示为 RUS，操作结果生成一个新的关系，其元组由属于 R 的元组和属于 S 的元组共同组成。本题中，执行并操作后，如果 S 中的 10 个元组在 R 中都有，则结果最小为“15”若 S 中的 10 个元组在 R 中都没有，则结果最大为“25”。因此，本题填  $15 \sim 25$ 。  
94. 盲打是指在电脑上打字的时候不看键盘。盲打的正规指法:五指微下屈, 两个大拇指控制空格键, 左手另四指分别放在 A、S、D、F 四键上, 右手另四指分别放在 J、K、L、;四个键上, 这样左右两个食指就分别放在 F 键和 J 键。因此, 本题填  $\underline{\mathbf{J}}$  。  
95.本题属于基本操作，填注销。

96.全选操作的快捷键是“Ctrl+A”。因此，本题填Ctrl+A。  
97. 有多种类型的“软键盘”，可以利用“软键盘”快速输入各种符号。因此，本题填软键盘。  
98. 若在一个公式中用到一个或多个单元格地址，则认为该公式引用了单元格地址。根据不同的需要，在公式中引用单元格地址分为3种引用方式，即相对地址引用绝对地址引用和混合地址引用。因此，本题填混合地址。  
99.在Excel中，要冻结某行，选定下一行，执行“窗口”菜单中的“冻结窗格”命令。要冻结某列，选定右侧一列，执行“窗口”菜单中的“冻结窗格”命令。本题中，要冻结第一行应该选定第二行(行号为“2”)；要冻结第一列，应该选定第二列(列标为“B”)。因此，本题填B2。  
100. 在 PowerPoint 中，使用幻灯片母版可以定义每张幻灯片共同具有的一些统一特征。因此，本题填幻灯片母版。
