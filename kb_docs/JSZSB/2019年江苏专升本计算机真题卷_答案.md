# 四、填空题

71. 基数是指各种进位记数制中允许选用基本数码的个数。数制中每一固定位置对应的单位值称为位权。R 进制数，整数部分最低位的位权是  $R^{0}$ ，每往高去一位，位权的幂次加 1；小数部分最高位的位权是  $R^{-1}$ ，每往低去一位，位权的幂次减 1。因此，本题填  $\underline{16} 、 \underline{16}^{3}$ 。

72.本题属于基本常识，填CPU、内存条。

73.数据的逻辑结构可以分为：线性结构和非线性结构。线性结构的代表有：线性表、栈、

队列、字符串、数组、广义表。非线性结构的代表有：树结构（树、二叉树）、森林、图结构（有向图、无向图）。因此，本题填树、图，两个空所填内容的顺序可以调换。

74. 明文是未加密的报文，密文是加密后的报文。根据题目含义：恺撒密码的加密思想是将每一个英文字母替换为字母表中排列在其后的第 K 个字母。若  $\mathrm{K} = 3$ ，则明文“hello”加密后的密文为“khoor”，即明文中的每个字母都替换为字母表中排列在其后的第 3 个字母；若  $\mathrm{K} = 5$ ，加密后得到的密文为“firns”，则明文为“admin”，即密文中的每个字母都替换为字母表中排列在其前的第 5 个字母。因此，本题填 khoor、admin。

75.音频信号数字化的过程：取样、量化、编码。因此，本题填量化、编码。

76.SQL语言提供了SELECT语句进行数据库查询。SELECT查询语句简单的形式为：

SELECT A1, A2,..., An (指出查询结果表的列名，相应于投影操作)

FROM R1, R2,..., Rm (指出基本表或视图，相应于连接操作)

WEHEREF

(F 为条件表达式，相应于选择操作的条件)

如果查询时有多个条件需要同时包含的话，则多个条件之间用“and”连接。因此，本题填FROM、and。

77. 安装操作系统和软件之前，首先需要对硬盘进行分区和格式化，然后才能使用硬盘保存各种信息。磁盘分区是使用分区编辑器在磁盘上划分几个逻辑部分。格式化是指对磁盘或磁盘中的分区进行初始化的一种操作，这种操作通常会导致现有的磁盘或分区中所有的文件被清除。因此，本题填分区、格式化。

78.在Word中，段落主要有四种缩进方式：首行缩进、悬挂缩进、左缩进和右缩进。首行缩进是将段落的第一行从左向右缩进一定的距离，首行外的各行都保持不变。悬挂缩进是指段落的首行文本不加改变，而除首行以外的文本缩进一定的距离。左缩进是设置整个段落左端距离页面左边界的起始位置。右缩进是指段落右端距离页面左边起始位置。因此，本题填首行、悬挂。

79. 第1空：MAX函数的功能是求各参数的最大值。本空中所求最大值的区域范围是“B2:B8”。因为要通过拖动填充柄复制至D9单元格，在Excel中，复制公式的时候，遵循“相对地址引用随着目标单元格的变化而变化，绝对地址引用不变”的原则，所以本空使用相对地址引用。第2空：MIN函数的功能是求各参数的最小值。本空中所求最大值的区域范围是“B2:B8”。因为要通过拖动填充柄复制至D9单元格，在Excel中，复制公式的时候，遵循“相对地址引用随着目标单元格的变化而变化，绝对地址引用不变”的原则，所以本空使用相对地址引用。第3空：IF函数的形式为：IF(P,T,F)，其中P是能产生逻辑值（TRUE或FALSE）的表达式，T、F分别都是表达式。函数的功能：若P为真，则返回结果为T表达式的值；否则，返回结果为F表达式的值。因此，本题填=MAX（B2:B8）、=MIN（B2:B8）、=IF(E2>=240,"A","B")（或者填：=IF（E2<240,"B","A")）。

80.在 PowerPoint 中，使用“动画刷”按钮可以快速地复制动画效果。因此，本题填动画刷。
