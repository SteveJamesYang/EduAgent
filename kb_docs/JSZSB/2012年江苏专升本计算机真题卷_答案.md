# 四、填空题

81. 原码、反码和补码都是带符号数，默认用 8 位来表示，其中最高位(第 1 位)用作符号位，正数的符号位是“0”，负数的符号位是“1”。整数的原码是指其符号位(最高位)的 0 或 1 表示正或负，其数值部分就是绝对值的二进制表示。整数的反码是相对于原码来说的，正整数的反码和原码相同，负整数的反码是其对应的原码符号位保持不变，面数值部分按位取反。正整数的补码与其原码一致：负整数的补码是其对应的反码在最后一位加 1。本题中，先将十进制数“29”转换为 7 位二进制数 0011101。再将 7 位二进制数 011101“先取反，再加“1”，得到二进制数 1100011。最高位符号位“1”表示负数。因此，本题填 11100011。

82. 每个 ASCII 码字符在传输时采用 8 位。本题计算过程为:  $9600 / 8 * 60 = 72000$  先计算出每秒钟传输的 ASCII 码字符数, 再乘以 60 , 算出每分钟 (60 秒) 传输的 ASCII 码字符数, 因此, 本题填 72000 。

98.在Excel中,若在一个公式中用到一个或多个单元格地址,则认为该公式引用了单元格地址。根据不同的需要,在公式中引用单元格地址分为3种引用方式,即相对地址引用、绝对地址引用和混合地址引用。相对引用包含了当前单元格与公式所在单元格的相对位置。默认设置下,Excel使用的都是相对引用。绝对地址引用需在列标和行号前都加上$符号,如$D$1(绝对地址,行列均固定)。混合引用是指在一个单元格地址引用中既有相对引用,也有绝对引用。它包括两种情况,一种是绝对列相对行,即$AI、$B2等形式;另一种情况是相对列绝对行,即A$1、B$2等形式。因此,本题填绝对。

"视图"工具栏上的“幻灯片放映”按钮的功能是从当前幻灯片开始放映，相当Shift+F5。因此，本题填Shift+F5。
