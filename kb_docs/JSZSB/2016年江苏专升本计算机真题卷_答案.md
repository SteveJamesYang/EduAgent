# 四、填空题

71. 十进制数转换为二进制数的转换分为两步: 整数部分采用除基取余法 (山坡法则): “除以基取余, 先得到余数为低(位)后得到的余数为高(位)”; 小数部分采用乘基取整法: “乘基取整, 先整为高(位), 后整为低(位)”。八进制数转换为二进制数把每 1 位八进制数转换为对应的 3 位二进制数。二进制数转换为十六进制数: 以小数点为界, 将整数部分自右向左和小数部分自左向右分别按每 4 位为一组, 不足 4 位用 0 补足, 然后将各个 4 位二进制数转换为对应的 1 位十六进制数。不同进制的数之间不能直接相加, 必须要转换为同一进制之后才能进行运算。因此, 本题填 “100010.101” “100111.101” “1001010.010” “4A.4”。

72. 若某处理器可以访问内存空间的最大地址为 FFFFFFFH，则内存容量=末地址-首地址+1=FFFFFF000000000 转换到十进制数为  $1 \times 16^{*}$ 。在内存储器中，每个存储单元可以存放 1 个字节，本题的存储单元为  $1 \times 16^{*}$ ，所以存储空间为  $(1 \times 16^{*}) B = (22) B = 4$  GB。微机硬件系统中，地址总线的位数决定了 CPU 可以访问的存储器最大空间。如果 CPU 的地址线宽度为 n，那么 CPU 的寻址大小是  $2^{n} (B)$ 。因此，本题中处理器的地址总线有 32 根，处理器的寻址能力为 4GB。因此，本题填“32”“4”。  
73. 第1小题的第1空:路由器右侧主机的IP地址为“192.18.6.10”, IP地址首字节为“192”,可以判断出这是C类IP地址。第1小题的第2空:子网掩码是一个32位的代码, 其中与IP地址中网络号对应位置处的二进位是“1”, 与主机号对应位置处的进位是“0”。对于A类地址来说, 默认的子网掩码是255.0.0.0;对于B类地址来说, 默认的子网掩码255.255.0.0;对于C类地址来说, 默认的子网掩码是255.255.255.0。第1小题的第3空:一个IP地址包含“网络号”和“主机号”两个部分, 对于C类IP地址而言, 前3个字节是“网络号”, 最后1个字节是“主机号”, 接口B的IP地址为“192.18.0.254”, 属于C类IP地址, 因此主机号是“254”。第2小题:路由器端口的IP地址与相连子网的IP地址具有相同的网络号。与路由器接口A相连的主机A的IP地址为“192.18.26.10”, 属于C类IP地址, 前3个字节是网络号, 另外题目中已经说明“路由器接口A和B的IP地址中主机号相同”, 所以路由器接口A的IP地址为“192.18.26.254”。因此, 本题填“C”、“255.255.255.0”、“254”、“192.18.26.254”。  
74. 图像数据量的计算公式:图像数据量(B)=图像水平分辨率 X 图像垂直分辨率 X 像素深度 (b)/8。第 1 空:图像数据量=1280*1024*24/8. 再将得到的结果转换为以 MB 为单位即除以 (1024*1024), 得到最终结果 3.75 MB。第 2 空: 一幅图像的颜色(亮度)的最大数目=2 像素深度。因为颜色总数为 256, 所以像素深度为 8 。第 3 空: 图像数据量=1024*768*8/8, 再将得到的结果转换为以 1 KB 为单位即除以 1024, 得到最终结果 768 KB。第 4 空:3.75 MB 是 768 KB 存储空间的 5 倍计算方法:3.75X 1024/768。因此, 本题填 “3.75” “8” “768” “5”。  
75. 操作系统的存储管理功能是管理内存资源的高效、合理使用。现在，操作系统一般都采用虚拟存储技术(也称虚拟内存技术，简称虚存)进行存储管理。虚拟存储技术是把内存和外存(硬盘)结合起来管理，为用户提供一个容量比实际内存大得多的“虚拟存储器”。因此，本题填“虚拟”。

77. 在 Windows 中通过组合键“Shift+Delete”可以将硬盘中选定的文件或文件夹彻底删除，而不经过回收站。因此，本题填"Shift"。  
78. 在 Word 中，缩进是指段落的两端到页面左右边距的距离，有四种缩进设置首行缩进、悬挂缩进、左缩进和右缩进。“首行缩进”是将段落的第行向右缩进定的距离，其他行位置是不变的。“悬挂缩进”是将除段落第一行以外的所有行向右缩进定的距离，第一行的位置是不变的。“左缩进”是将整个段落向右缩进定的距离。“右缩进“"是将整个段落向左缩进定的距离。因此本题填"首行”。

79. 在Excel中,复制公式的时候,遵循“相对地址引用随着目标单元格的变化而变化,绝对地址引用不变”的原则。E5中的公式“=\\(A1-A\\)2-B1+ \\(B\\)2”复制到F6,目标单元格“行向下一行,列向右一列”,即“\\(A1”变为“\\)A2”,“A\\(2”变为“B\\)2”,“B1”变为“C2”,“\\(B\\)2”不变。因此,本题填“=\\(A2-B\\)2-C2+ \$B\\)2”。  
80. 已创建的演示文稿在没有安装 PowerPoint 软件的计算机播放，可以将演示文稿进行打包操作。因此，本题填“打包”。
