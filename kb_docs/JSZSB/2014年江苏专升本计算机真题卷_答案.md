# 四、填空题

81. 原码、反码和补码都是带符号数，默认用 8 位来表示，其中最高位(第 1 位)用作符号位，正数的符号位是 0，负数的符号位是 1。因此，本题填  $\underline{1}$ 。
82. 十进制数转换为二进制数时，由于整数和小数的转换方法不同，所以先将十进制数的整数部分和小数部分分别转换后，再加以合并。其中，整数部分转换采用“除基取余”法，小数部分转换采用“乘基取整”法。本题中，整数部分十进制数 89 转换为二进制数 1011001。小数部分 0.625 转换为二进制数 0.101，合并后的结果为“1011001.101”。因此，本题填 1011001.101。  
83. 显示器的分辨率是指整屏最多可显示像素的多少，一般用“水平分辨率×垂直分辨率”来表示。因此，本题填像素。  
84. USB 的版本: 1.0 版(速率:  $1.5 \mathrm{Mb} / \mathrm{s}$ ); 1.1 版(速率:  $1.5 \mathrm{MB} / \mathrm{s}$ ); 2.0 版(速率:  $60 \mathrm{MB} / \mathrm{s}$ ); 3.0 版(速率:  $400 \mathrm{MB} / \mathrm{s}$  )。因此, 本题填  $\underline{\mathbf{3 . 0}}$  。  
85.总线上通常有三类信号:数据信号地址信号和控制信号，负责传输这些信号的线路分别称为数据总线、地址总线和控制总线。因此，本题填控制。  
86.程序是告诉计算机做什么和如何做的一组(指令)语句。这些指令(语句)都是计算机所能理解并能够执行的一些命令。因此，在计算机内部，程序是由一连串指令组成的。指令是构成程序的基本单位。因此，本题填程序。  
87. 一般情况下高级语言的运行步骤是:①上机输人和编辑源程序;②对源程序进行编译, 产生目标程序;③进行连接处理, 产生可执行程序;④运行可执行程序得到运行结果。因此, 本题填目标程序。  
88. 误码率是数据传输中出错数据占被传输数据总数的比例，它是评价数据传输可靠性的指标。因此。本题填误码率。  
89.全球可以有  $2^{x} - 2^{-} = 126$  个网络获得A类地址，可以有  $2^{14} = 16384$  个网络获得B类地址。因此，本题填126。  
90.移动通信是指处于移动状态的对象之间的通信，也是微波通信的一种。最有代表性的移动通信是手机一个人移动通信系统。它由移动台(手机)、基站、移动电话交换中心等组成。因此，本题填基站。  
91. 为了不产生失真，按照取样定理取样频率不应低于声音信号最高频率的两倍。因此，语音的取样频率一般为  $8 \mathrm{kHz}$  。音乐的取样频率应在  $40 \mathrm{kHz}$  以上。因此，本题填  $40 \mathrm{kHz}$  。  
92. 一幅图像的颜色(亮度)的最大数目。本题中，颜色总数为 256，因此像素深度为 8。因此，本题填 8。  
93. 实体一般具有若干特征，用于描述其性质这种特征称之为实体的属性。每个属性有其取值的范围(值域)。因此，本题填值域。  
94.本题属于基本概念填内存。  
95. Alt+Print Screen: 将当前活动窗口复制到剪贴板; Print Screen; 将当前整个桌面复制到剪贴板。因此，本题填剪贴板。  
96. 在 Word 中, "Ctrl+End" 和 "Ctrl+Home" 这两个组合键的作用分别是: 将光标移到文档末尾/将光标移到文档开头。因此，本题填 Ctrl+End。  
97. 在 Word 中缩进是指段落的两端到页面左右边距的距离，有四种缩进设置:首行缩进悬挂缩进.左缩进和右缩进。“首行缩进”是将段落的第一行向右缩进一定的距离，其他行位置是不变。“悬挂缩进”是将除段落第一行以外的所有行向右缩进一定的距离，第一行的位置是不变的。“左缩进”是将整个段落向右编进定的距离。“右缩进”是将整个段落向左缩进一定的距离。因此，本题填悬挂缩进。
98. 删除D列后，A.B.C三列位置不变，原本D列右侧的列(从E列)开始依次往左移，填补空缺的位置。即E列变为D列，F列变为E列，G列变为P列……依次类推。在该种情况下，只要行列位置有变化，不管公式中是相对地址引用，还是绝对地址引用，相应的地址都要发生改变。因此，本题填=D3+$D$2。  
99.COUNTIF(A1, A2)函数的功能统计 A1 指定的数据区域内，满足 A2 给定条件的单元格的个数。本题是在“F3:H7”的区域中，求出值“>12”的单元格的个数。因此，本题填 6。  
100.在 PowerPoint 中，选择一种自动版式后，用虚线框圈起的指明“单击此处添加标题或文本”的地方即是“文本占位符”。因此，本题填版式。
