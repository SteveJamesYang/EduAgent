# 三、填空题

61.本题属于基本概念，填  $0 \sim 2^{n} - 1$ 。  
62. 原码反码和补码都是带符号数，默认用 8 位来表示，其中最高位(第 1 位)用作符号位，正数的符号位是“0”，负数的符号位是“1”。整数的原码是指其符号位(最高位)的 0 或 1 表示 X 的正或负，其数值部分就是该数绝对值的二进制表示。整数的反码是相对于原码来说的，正整数的反码和原码相同，负整数的反码是其对应的原码符号位保持不变，而数值部分按位取反。正整数的补码与其原码一致：负整数的补码是其对应的反码在最后一位加 1。本题中，通过最高位(第 1 位)可以知道 X 是负数。将除去符号位以外的数先减 1，再求反，再转换到十进制。可以得到 X 为“-104”：通过最高位(第 1 位)可以知道 Y 是正数，除去符号位，转换到十进制，可以得到 Y 为“+51”。两者相加，结果为“-53”。因此，本题填 -53。  
63.迄今为止，我们所使用的计算机是按照美籍匈牙利数学家冯·诺依曼提出的“存储程序控制”的原理进行工作的。因此,本题填存储程序控制原理。  
64. 芯片组一般由两块超大规模集成电路组成:北桥芯片和南桥芯片。北桥芯片是存储控制中心, 用于高速连接 CPU、内存条、显卡, 并与南桥芯片互联。南桥芯片是 I/O 控制中心, 主要与 PCI 总线槽、USB 接口、硬盘接口、音频编解码器、BIOS、CMOS 存储器等连接并借助 Super I/O 芯片提供对键盘、鼠标、串行口和并行口等的控制。因此,本题填存储。  
65. 操作系统的主要功能:①多任务处理与处理器管理对一CPU资源的管理;②存储管理一对内存资源的管理;③文件管理对硬盘资源的管理;④设备管理一对I/O设备的管理。因此,本题填文件管理。  
66.算法的有穷性是指一个算法总是在执行了有限步的操作之后终止。因此,本题填有穷性。  
67. 互联网传输层,规定了怎样进行端对端的数据传输。有 TCP 和 UDP 两个协议。大部分应用程序使用 TCP 协议, 它负责可靠地完成数据从发送计算机到接收计算机的传输。如电子邮件的传送和网页的下载等; 而使用 UDP 协议时, 网络只是尽力面为地进行快速数据传输。但不保证传输的可靠性, 例如音额和视频数据的传输就采用了 UDP 协议。因此, 本题填 UDP。

68.时分多路复用(TDM)是数字通信的主要手段，把一个传输通道进行时间分割以传送若干话路的信息，把N个话路设备接到一条公共的通道上，按一定的次序轮流给各个设备分配一段使用通道的时间。当轮到某个设备时，这个设备与通道接通执行操作。与此同时，其他设备与通道的联系均被切断，待指定的使用时间间隔到，则通过时分多路转换开关把通道连接到下一个要连接的设备上去。因此，本题填时分。  
69.本题属于基本概念，填 Flash。  
70. 数据库中存在“元数据”, 它们是有关数据库的定义信息如数据类型、模式结构、使用权限等, 这些数据的集合称为数据字典。因此, 本题填数据字典。  
71. 按组合键"Ctrl+Alt+Del"可以弹出“任务管理器”对话框,在该对话框中可以强制结束未响应的程序。因此,本题填Ctrl+Alt+Del。  
72.本题属于基本操作填启动。  
73.在Word中,可以在文档中插入其他应用程序所创建的对象(例如:插入Excel工作表),操作方法:执行“插入”菜单中的“对象”命令。因此,本题填对象。  
74.在Word中输入文本的方式有两种：①插入方式(系统的默认状态)；②改写方式(新键人的字符将替代当前光标位置后的字符)。因此，本题填“改写”。  
75.本题属于基本操作，填格式。  
76.在Excel中输人数据时,若在某正数数值两边加一个小括号，则显示该正数数值的负数形式。因此，本体填8。  
77. IF函数的形式为:IF(P,T,F),其中P是能产生逻辑值(TRUE或FALSE)的表达式,T、F分别都是表达式。函数功能:若P为真,则返回结果为T表达式的值;否则,返回结果为F表达式的值。本题属于函数嵌套问题,先解里层的函数,再解外层的函数。因此,本题填TRUE。  
78. 删除第 3 行后,原有第 3 行下方的行依次往上填补空缺。即原有第 4 行变为第 3 行,原有第 5 行变为第 4 行......依次类推。在该种情况下,只要行列位置有变化,不管公式中是相对地址引用,还是绝对地址引用相应的地址都要发生改变。因此,本题填=AVERAGE( $ AI,B2:B3,$C$4)。  
79. 在 PowerPoint 中，执行“插入”菜单中的“新幻灯片”命令，可以插入张新幻灯片，新插入的幻灯片位于选定幻灯片的后面。因此,本题填  $\underline{\mathbf{3}}$  。  
80.本题属于基本概念，填占位符。
