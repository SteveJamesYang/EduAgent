# 三、填空题

61. 现代信息技术的主要特征：以数字技术为基础，以计算机及其软件为核心，采用电子技术（包括激光技术）进行信息的收集、传递、加工、存储、显示与控制。因此，本题填数字。  
62. 为了提高线路利用率，总是设法在一条传输线路上，传输多个模拟信号或数字信号，这就是多路复用。因此，本题填多路复用。

63.打印机的主要性能指标有：

- 打印精度：打印机的分辨率，它用dpi（每英寸可打印的点数）来表示，是衡量图像清晰程度最重要的指标。

- 打印速度：每秒打印的字符数目（针式打印机）；每分钟打印的页数（激光打印机、喷墨打印机）。

·色彩表现能力：打印机可打印的不同颜色的总数。

- 其它：包括打印成本、噪音、可打印的幅面大小、功耗及节能指标、可打印的拷贝数目、与主机的接口类型等。

因此，本题填精度。

64.本题属于基本概念，填存取时间。

65. 在 Windows 操作系统中，虚拟存储器是由物理内存（主板上的 RAM）和硬盘上的虚拟内存（“交换文件”）联合组成的，页面的大小是 4KB。交换文件的文件名是 pagefile.sys，它位于操作系统的根目录下。

因此，本题填 pagefile.sys。

66.著名的瑞士计算机科学家尼克劳斯.沃思关于“算法”提出了两个著名的论述：①数据结构+算法=程序；②计算机科学就是研究算法的学问。因此，本题填算法。

67. 一个 A 类网络最多可容纳  $2^{24} - 2$  即 16777214 台主机, 一个 B 类网络最多可容纳  $2^{16} - 2$  即 65534 台主机, 一个 C 类网络最多可容纳  $2^{8} - 2$  即 254 台主机。因此, 本题填  $\underline{254}$  。

68.IP协议的第4版（简称IPv4）规定，每个IP地址是一个32位二进制数。但IPv4只有大约36亿个地址，估计很快将被分配完毕。IP协议第6版（IPv6）已经把IP地址的长度扩展到128位。因此，本题填128。

69.本题属于基本概念，填YUV。

70.本题属于基本概念，填视图。

71. Windows 附件中的“记事本”程序所编辑处理的文本就是简单文本。默认状态下，系统将启动“记事本”程序打开.txt文件。因此，本题填记事本。

72.本题属于基本概念，填255。

73. 宏是一系列 Word 命令和指令，这些命令和指令组合在一起，形成了一个单独的命令，以实现任务执行的自动化。执行宏就是顺序执行它所包含的全部操作。因此，本题填宏。

74. 在Word中，正常情况下，段落内的每一行都是根据页面宽度自动换行。若需要在段落内的某个位置强制换行，但不拆分段落，则可插入换行符（Shift+Enter）。因此，本题填换行符。

75. Word 文档窗口可以拆分为两个窗口，利用窗口拆分可以将一个大文档不同位置的两个部分显示在两个窗口中，即可以同时看到一个长文档不同页面上的内容，从而可以很方便地编辑文档。操作方法 1：使用“窗口”下拉菜单中的“拆分”命令；操作方法 2：拖动垂直滚动条上端的小横条拆分窗口。因此，本题填拆分。

76.在第2列（B列）之前插入一列，新插入的列成为第2列（B列），原有的第2列（B列）

及其后方的列依次往右一列，即原有第2列（B列）变为第3列（C列），原有第3列（C列）变为第4列（D列）……依次类推。在该种情况下，只要行列位置有变化，不管公式中是相对地址引用，还是绝对地址引用，相应的地址都要发生改变。因此，本题填=A1+$C2+$D$3。  
77. 在函数中有两个单元格地址引用，分别是“A1：A2”和“B1：C2”，两个区域使用空格隔开，表示求这两个区域的公共单元格。可以看出，这两个区域并没有公共单元格。因此，本题填#NULL!。错误值“#NULL！”的含义是“试图为两个并不相交的区域制定交叉点”。  
78. 选定某单元格，使之成为当前单元格后，按“Delete”键只清除单元格中的数据内容。本题中，原有单元格中的格式，即“日期”格式被保留。在默认情况下，Excel 是 1900 日期系统，约定 1900 年 1 月 1 日是数字序列号“1”。因此，本题填 1900 年 1 月 1 日。  
79. 新建演示文稿后，默认第一张幻灯片是“标题版式”的幻灯片，以后插入新幻灯片默认是“标题和文本版式”。因此，本题填标题和文本。  
80.在PowerPoint中，“放映类型”主要包括如下几个选项：

- 演讲者放映（全屏幕）：该选项表示以全屏幕形式放映幻灯片。

·观众自行浏览（窗口）：该选项表示放映的幻灯片出现在窗口中。

·在展台浏览（全屏幕）：该选项表示以全屏幕形式在展台上演示幻灯片。这种放映方式不允许用户在幻灯片放映过程中进行干预，因此除了保留鼠标指针用于选择屏幕对象之外，不能对幻灯片做其他任何控制。

因此，本题填在展台浏览。
