# 四、填空题

71. 答案：减法

解析:本题所述中之所以补码运算可以将减法运算和加法运算统一起来,是因为可以利用高位溢出的情况,将减法转换成加法,从而可以简化运算,实现其运算统一性。

72. 答案: 运算器、控制器

解析：运算器，对数据进行各种算术运算和逻辑运算。控制器，CPU的指挥中心，协调机器各个部件工作。(结合2018年/题74图理解)。

73. 答案：运算成分、传输成分

解析：程序设计语言基本成分主要由：数据成分，描述程序所处理的数据对象(数据名、数据类型、数据结构)。运算成分，描述程序所包含的运算(算术运算、逻辑运算)。控制成分，表达程序中的控制构造(if条件语句、循环语句)。传输成分，表达程序中的数据传输(赋值语句、输入/输出语句)。

74. 答案：语义、语法

解析：一个网络协议由以下三个要素组成：语法，用来规定用户数据与控制信息的结构与格式以及数据出现的顺序。语义，用来说明通信双方应当如何做，需要发出何种控制信息及完成的动作与做出的响应。时序(同步)，对通信实现顺序的详细说明。
